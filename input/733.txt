title="Odda">Odda</a></div></div></div><p><b>Industri</b> er ei form for økonomisk verksemd som er kjenneteikna av at</p><ul><li>Råvarer vert omskapte til ferdige produkt.</li><li>Produksjonen går føre seg i stor skala og ved hjelp av maskiner og kraft utanfrå.</li><li>Store førehandsinvesteringar er naudsynt om det skal verte utbytte av det.</li></ul><p>Industrien er den andre sektoren i ei oppdeling av heile det økonomiske livet i tre sektorar: Primærnæringane er råstoffutvinning, <a href="/wiki/Gruvedrift" title="Gruvedrift">gruvedrift</a>, <a href="/wiki/Landbruk" title="Landbruk">landbruk</a> og <a href="/wiki/Fiske" title="Fiske">fiske</a>. Sekundærnæringane er industriverksemd og tertiærnæringane er <a href="/wiki/Handel" title="Handel">handel</a>, <a href="/w/index.php?title=Service&amp;action=edit&amp;redlink=1" class="new" title="Service (sida finst ikkje)">service</a> og <a href="/wiki/Transport" title="Transport">transport</a>.</p><p>Det finst ulike typar industri, så som råvareindustri, <a href="/wiki/Tekstil" title="Tekstil">tekstilindustri</a>, <a href="/wiki/Bil" title="Bil">bilindustri</a> osb.</p><p>I vidare nemning er industri all slags næringsliv - som til dømes underhaldningsindustri eller <a href="/wiki/Film" title="Film">filmindustri</a>.</p><p>Dei mest vellukka verksemdene innanfor ei industrigrein har det med å vera anten dei selskapa som vart starta med ein stor innsatskapital, eller tidlege inovatørar som var fyrst ute med å få ein ny teknologi ut på marknaden.</p><p>Industrien vart eit hovudområde i næringslivet i <a href="/wiki/Europa" title="Europa">Europa</a> og <a href="/wiki/Nord-Amerika" title="Nord-Amerika">Nord-Amerika</a> under <a href="/wiki/Den_industrielle_revolusjon" class="mw-redirect" title="Den industrielle revolusjon">den industrielle revolusjonen</a>. Gjennom ei rask utvikling av nye teknologiar, så som mekaniske <a href="/wiki/Vevstol" title="Vevstol">vevstolar</a> og <a href="/wiki/Dampmaskin" title="Dampmaskin">dampmaskiner</a>, vart dei gamle <a href="/wiki/F%C3%B8ydal" class="mw-redirect" title="Føydal">føydale</a> og <a href="/w/index.php?title=Merkantil&amp;action=edit&amp;redlink=1" class="new" title="Merkantil (sida finst ikkje)">merkantile</a> økonomiane kasta om.</p><p>I Noreg har det vore foredla trevirke i sagbruk i fleire hundreår, men dette vert ikkje rekna som eigentleg industri. Den norske industrialiseringa tok til så smått i <a href="/wiki/1815" title="1815">1815</a>, heldt fram på <a href="/wiki/1840" title="1840">1840</a>-talet, med etableringa av <a href="/w/index.php?title=Spinneri&amp;action=edit&amp;redlink=1" class="new" title="Spinneri (sida finst ikkje)">spinneri</a> og <a href="/w/index.php?title=Veveri&amp;action=edit&amp;redlink=1" class="new" title="Veveri (sida finst ikkje)">veveri</a> som nytta mekanisk kraftoverføring frå <a href="/wiki/Elv" title="Elv">elver</a> og <a href="/wiki/Foss" title="Foss">fossefall</a>, men skaut fart for alvor først etter hundreårsskiftet, basert på <a href="/wiki/Hydroelektrisk_energi" class="mw-redirect" title="Hydroelektrisk energi">hydroelektrisk energi</a>.</p><p>Eit <a href="/wiki/Samfunn" title="Samfunn">samfunn</a> der industri er dominerande, vert kalla eit <a href="/wiki/Industrisamfunn" title="Industrisamfunn">industrisamfunn</a>. Slike samfunn har vore gjennom ein prosess med <a href="/wiki/Industrialisering" title="Industrialisering">industrialisering</a>. Eit samfunn der industri har vorte mindre viktig, kan kallast eit <a href="/wiki/Postindustrielt_samfunn" title="Postindustrielt samfunn">postindustrielt samfunn</a>.</p><table cellspacing="0" class="navbox" style="border-spacing:0;"><tr><td style="padding:2px;"><table cellspacing="0" class="nowraplinks hlist navbox-inner" style="border-spacing:0;background:transparent;color:inherit;"><tr><th scope="row" class="navbox-group"><a href="/wiki/Autoritetsdata" title="Autoritetsdata">Autoritetsdata</a></th><td class="navbox-list navbox-odd" style="text-align:left;border-left-width:2px;border-left-style:solid;width:100%;padding:0px;"><div style="padding:0em 0.25em;"><ul><li><a rel="nofollow" class="external text" href="http://d-nb.info/gnd/4026779-9">GND</a></li><li><a rel="nofollow" class="external text" href="http://id.ndl.go.jp/auth/ndlna/00570111">NDL</a></li></ul></div></td></tr></table></td></tr></table><!-- NewPP limit reportParsed by mw1300Cached time: 20180319222903Cache expiry: 1900800Dynamic content: falseCPU time usage: 0.044 secondsReal time usage: 0.053 secondsPreprocessor visited node count: 22/1000000Preprocessor generated node count: 0/1500000Post‐expand include size: 1233/2097152 bytesTemplate argument size: 0/2097152 bytesHighest expansion depth: 4/40Expensive parser function count: 0/500Unstrip recursion depth: 0/20Unstrip post‐expand size: 0/5000000 bytesLua time usage: 0.020/10.000 secondsLua memory usage: 885 KB/50 MB--><!--Transclusion expansion time report (%,ms,calls,template)100.00%   35.235      1 Mal:Autoritetsdata100.00%   35.235      1 -total--></div><!-- Saved in parser cache with key nnwiki:pcache:idhash:2701-0!canonical and timestamp 20180319222903 and revision id 2802701 --><noscript><img src="//nn.wikipedia.org/wiki/Special:CentralAutoLogin/start?type=1x1" alt="" title="" width="1" height="1" style="border: none;
