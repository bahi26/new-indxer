ASV</div></div></div><p><b>Rūpniecība</b> jeb <b>industrija</b> (no <a href="/wiki/Lat%C4%AB%C5%86u_valoda" title="Latīņu valoda">latīņu</a>: <span lang="la" xml:lang="la"><i>industria</i></span> — ‘rosība, darbība’<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup>) ir viena no svarīgākajām <a href="/wiki/Tautsaimniec%C4%ABba" class="mw-redirect" title="Tautsaimniecība">tautsaimniecības</a> nozarēm, kura ietver dažādu <a href="/wiki/Prece" title="Prece">preču</a> ražošanu. Būtībā tas ir <a href="/wiki/Uz%C5%86%C4%93mums" title="Uzņēmums">uzņēmumu</a> kopums (<a href="/wiki/R%C5%ABpn%C4%ABca" title="Rūpnīca">rūpnīcas</a>, <a href="/wiki/Fabrika" class="mw-redirect" title="Fabrika">fabrikas</a>, <a href="/wiki/Raktuves" class="mw-redirect" title="Raktuves">raktuves</a>, <a href="/wiki/Elektrostacija" title="Elektrostacija">elektrostacijas</a> un citi), kuros veic šo preču ražošanu. Savukārt iegūtās preces nodrošina tālāku citu tautsaimniecību attīstību. Vēl pie rūpniecības pieskaita <a href="/wiki/Der%C4%ABgais_izraktenis" class="mw-redirect" title="Derīgais izraktenis">derīgo izrakteņu</a> iegūšanu, <a href="/wiki/Ener%C4%A3ija" title="Enerģija">enerģijas</a> ražošanu, <a href="/wiki/Me%C5%BEs" title="Mežs">mežu</a> apsaimniekošanu un citas nozares, tai skaitā arī <a href="/wiki/Pakalpojums" title="Pakalpojums">pakalpojumus</a>.</p><p>Katras <a href="/wiki/N%C4%81cija" title="Nācija">nācijas</a> turība galvenokārt balstās uz tās rūpniecību. Par industriālākajām valstīm mēdz spriest pēc <a href="/wiki/Iek%C5%A1zemes_kopprodukts" title="Iekšzemes kopprodukts">iekšzemes kopprodukta</a> (IKP). Vislielākais IKP un, līdz ar to var pieņemt, visindustriālākās valstis ir <a href="/wiki/Amerikas_Savienot%C4%81s_Valstis" title="Amerikas Savienotās Valstis">Amerikas Savienotās Valstis</a>, <a href="/wiki/%C4%B6%C4%ABna" title="Ķīna">Ķīna</a>, <a href="/wiki/Jap%C4%81na" title="Japāna">Japāna</a>, <a href="/wiki/V%C4%81cija" title="Vācija">Vācija</a>, <a href="/wiki/Francija" title="Francija">Francija</a> un <a href="/wiki/Apvienot%C4%81_Karaliste" title="Apvienotā Karaliste">Apvienotā Karaliste</a>.</p><p></p><div id="toc" class="toc"><div class="toctitle" lang="lv" class="mw-editsection-divider"> | </span><a href="/w/index.php?title=R%C5%ABpniec%C4%ABba&amp;action=edit&amp;section=1" title="Labot sadaļu: Rūpniecības iedalījums">labot pirmkodu</a><span class="mw-editsection-bracket">]</span></span></h2><p>Rūpniecība tiek iedalīta trīs sektoros.<sup id="cite_ref-Tvtsv_2-0" class="reference"><a href="#cite_note-Tvtsv-2">[2]</a></sup> Pirmais ir <a href="/w/index.php?title=Prim%C4%81rais_sektors&amp;action=edit&amp;redlink=1" class="new" title="Primārais sektors (vēl nav uzrakstīts)">primārais sektors</a>, kuru veido <a href="/wiki/Lauksaimniec%C4%ABba" title="Lauksaimniecība">lauksaimniecība</a>, <a href="/wiki/Medniec%C4%ABba" title="Medniecība">medniecība</a>, <a class="mw-redirect" title="Zvejniecība">zvejniecība</a> un ieguves rūpniecība. Būtībā tās ir visas rūpniecības nozares, kur produkciju iegūst tieši no <a href="/wiki/Dabas_resursi" title="Dabas resursi">dabas resursiem</a>. Šis bija nozīmīgākais sektors līdz 19. gadsimta sākumam, kad sākās <a href="/wiki/R%C5%ABpniecisk%C4%81_revol%C5%ABcija" title="Rūpnieciskā revolūcija">Rūpnieciskā revolūcija</a>. Otrais ir <a href="/w/index.php?title=Sekund%C4%81rais_sektors&amp;action=edit&amp;redlink=1" class="new" title="Sekundārais sektors (vēl nav uzrakstīts)">sekundārais sektors</a>, kuru veido <a href="/w/index.php?title=Apstr%C4%81des_r%C5%ABpniec%C4%ABba&amp;action=edit&amp;redlink=1" class="new" title="Apstrādes rūpniecība (vēl nav uzrakstīts)">apstrādes rūpniecība</a>, <a href="/w/index.php?title=Energoapg%C4%81de&amp;action=edit&amp;redlink=1" class="new" title="Energoapgāde (vēl nav uzrakstīts)">energoapgāde</a>, <a href="/w/index.php?title=G%C4%81zes_apg%C4%81de&amp;action=edit&amp;redlink=1" class="new" title="Gāzes apgāde (vēl nav uzrakstīts)">gāzes apgāde</a>, <a href="/w/index.php?title=%C5%AAdensapg%C4%81de&amp;action=edit&amp;redlink=1" class="new" title="Ūdensapgāde (vēl nav uzrakstīts)">ūdensapgāde</a>, <a href="/wiki/Celtniec%C4%ABba" title="Celtniecība">celtniecība</a>, <a href="/w/index.php?title=Met%C4%81lapstr%C4%81de&amp;action=edit&amp;redlink=1" class="new" title="Metālapstrāde (vēl nav uzrakstīts)">metālapstrāde</a>, <a href="/wiki/Ma%C5%A1%C4%ABnb%C5%ABve" title="Mašīnbūve">mašīnbūve</a> un tā tālāk. Būtībā sekundārais sektors dalās <a href="/w/index.php?title=Viegl%C4%81_r%C5%ABpniec%C4%ABba&amp;action=edit&amp;redlink=1" class="new" title="Vieglā rūpniecība (vēl nav uzrakstīts)">vieglajā rūpniecībā</a> un <a href="/w/index.php?title=Smag%C4%81_r%C5%ABpniec%C4%ABba&amp;action=edit&amp;redlink=1" class="new" title="Smagā rūpniecība (vēl nav uzrakstīts)">smagajā rūpniecībā</a>. Visbeidzot trešais ir <a href="/w/index.php?title=Terci%C4%81rais_sektors&amp;action=edit&amp;redlink=1" class="new" title="Terciārais sektors (vēl nav uzrakstīts)">terciārais jeb pakalpojumu sektors</a>, kurā uzņēmumi sniedz <a href="/w/index.php?title=Pakalpojumi&amp;action=edit&amp;redlink=1" class="new" title="Pakalpojumi (vēl nav uzrakstīts)">pakalpojumus</a> saviem klientiem. Mūsdienās šis sektors pasaulē aizņem arvien lielāku daļu no <a href="/wiki/Iek%C5%A1zemes_kopprodukts" title="Iekšzemes kopprodukts">iekšzemes kopprodukta</a>. Viena no visstraujāk attīstītākajām terciārā sektora nozarēm ir <a href="/wiki/T%C5%ABrisms" title="Tūrisms">tūrisms</a>.</p><h2><span id="R.C5.ABpniec.C4.ABbas_nozares"></span><span
