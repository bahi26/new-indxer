id="Sensurile_.C3.AEn_care_este_folosit_cuv.C3.A2ntul_industrie"></span><span class="mw-headline" id="Sensurile_în_care_este_folosit_cuvântul_industrie">Sensurile în care este folosit cuvântul <i>industrie</i></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Industrie&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Modifică secțiunea: Sensurile în care este folosit cuvântul industrie">modificare</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Industrie&amp;action=edit&amp;section=1" title="Modifică secțiunea: Sensurile în care este folosit cuvântul industrie">modificare sursă</a><span class="mw-editsection-bracket">]</span></span></h2><p><b>Industria</b>, a devenit sectorul principal de producție al țărilor <a href="/wiki/Europa" title="Europa">europene</a> și <a href="/wiki/America_de_Nord" title="America de Nord">nord-americane</a> în timpul <a href="/wiki/Revolu%C8%9Bia_industrial%C4%83" title="Revoluția industrială">revoluției industriale</a>, care a răsturnat vechea economie negustorească și agrar-<a href="/wiki/Feudal" class="mw-redirect" title="Feudal">feudală</a> prin salturi rapide, succesive în tehnologie, așa cum au fost inventarea <a href="/wiki/Motor_cu_aburi" class="mw-redirect" title="Motor cu aburi">motorului cu aburi</a>, <a href="/w/index.php?title=R%C4%83zboi_mecanic_de_%C8%9Besut&amp;action=edit&amp;redlink=1" class="new" title="Război mecanic de țesut — pagină inexistentă">războiului mecanic de țesut</a> și cuceririle tehnice și tehnologice importante în producția la scară mare a <a href="/wiki/O%C8%9Bel" title="Oțel">oțelului</a> și <a href="/wiki/C%C4%83rbune" title="Cărbune">cărbunelui</a>. Țările industriale au adoptat o politică economină de tip <a href="/wiki/Capitalism" title="Capitalism">capitalist</a>. <a href="/wiki/Cale_ferat%C4%83" title="Cale ferată">Căile ferate</a> și <a href="/wiki/Vapor" title="Vapor">vapoarele</a> cu <a href="/wiki/Abur" title="Abur">aburi</a> au început să lege piețele foarte depărtate, dând posibilitatea noilor <a href="/wiki/Companie_(drept)" class="mw-redirect mw-disambig" title="Companie (drept)">companii private</a> să dezvolte niveluri de volum de afaceri și <a href="/wiki/Bog%C4%83%C8%9Bie" title="Bogăție">bogăție</a> nemaiîntâlnite până atunci.</p><p>În acest sens, industria se clasifică în:</p><ul><li><b>industria grea</b>, care se ocupă cu producerea mijloacelor de producție, și</li><li><b>industria ușoară</b>, care se ocupă cu producerea mărfurilor de larg consum.</li></ul><p>În alt sens, cuvântul <b>industrie</b> definește o grupare de <a href="/wiki/Afacere" title="Afacere">afaceri</a> care folosesc aceeași metodă pentru generarea profitului, cum ar fi "industria cinematografică", "industria construcțiilor de mașini", sau "industria cărnii și a produselor din carne", "<a href="/wiki/Minerit" title="Minerit">industria minieră</a>". Se referă de asemenea la anumite <a href="/wiki/Ramur%C4%83" class="mw-redirect mw-disambig" title="Ramură">ramuri</a> și o anumită zonă a producției concentrată pe <a href="/w/index.php?title=Fabricare&amp;action=edit&amp;redlink=1" class="new" title="Fabricare — pagină inexistentă">fabricare</a>, proces care implică mari investiții de <a href="/wiki/Capital_(economie)" title="Capital (economie)">capital</a> făcute mai înainte de a se obține <a href="/wiki/Profit" title="Profit">profituri</a>.</p><p>În <a href="/wiki/%C8%98tiin%C8%9Be_economice" title="Științe economice">științele economice</a> și în <a href="/wiki/Planificare_urban%C4%83" title="Planificare urbană">planificare urbană</a>, termenul <i>industrial</i> este sinonimul pentru modul de folosire intensiv al terenului destinat activităților economice care implica fabricarea și producția.</p><h2><span class="mw-headline"
