class="mw-body" role="main">			<a id="top"></a>			<div id="siteNotice" class="mw-body-content"><!-- CentralNotice --></div><div class="mw-indicators mw-body-content"></div><h1 id="firstHeading" class="firstHeading" lang="sco">Agricultur</h1>			<div id="bodyContent" class="mw-body-content">				<div id="siteSub" class="noprint">Frae Wikipedia</div>				<div id="contentSub"></div>								<div id="jump-to-nav" class="mw-jump">					Lowp tae:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">rake</a>				</div>				<div id="mw-content-text" lang="sco" dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><p><b>Agricultur</b> is the growin o <a href="/wiki/Fuid" title="Fuid">fuid</a> an ither things for <a href="/wiki/Fowk" title="Fowk">fowk</a> an <a href="/wiki/Ainimal" title="Ainimal">ainimals</a>. No juist <a href="/w/index.php?title=Fuid_crap&amp;action=edit&amp;redlink=1" class="new" title="Fuid crap (page disna exeest)">fuid craps</a> is grawn but things like <a href="/wiki/Flouer" title="Flouer">flouers</a>, ornamental <a href="/wiki/Plaunt" class="mw-redirect" title="Plaunt">plaunts</a> an nursery plaunts, <a href="/w/index.php?title=Manure&amp;action=edit&amp;redlink=1" class="new" title="Manure (page disna exeest)">manure</a> or dung, <a href="/w/index.php?title=Ainimal_hide&amp;action=edit&amp;redlink=1" class="new" title="Ainimal hide (page disna exeest)">ainimal hides</a>, <a href="/wiki/Laither" title="Laither">laither</a>, <a href="/w/index.php?title=Industrial_chemical&amp;action=edit&amp;redlink=1" class="new" title="Industrial chemical (page disna exeest)">industrial chemicals</a> (<a href="/wiki/Starch" class="mw-redirect" title="Starch">starch</a>, <a href="/wiki/Ethanol" title="Ethanol">ethanol</a>, an <a href="/wiki/Plastic" title="Plastic">plastics</a>), <a href="/wiki/Feebre" title="Feebre">feebres</a> (<a href="/wiki/Cotton" title="Cotton">cotton</a>, <a href="/wiki/Oo" title="Oo">oo</a>, <a href="/w/index.php?title=Hemp&amp;action=edit&amp;redlink=1" class="new" title="Hemp (page disna exeest)">hemp</a>, an <a href="/w/index.php?title=Flax&amp;action=edit&amp;redlink=1" class="new" title="Flax (page disna exeest)">flax</a>), <a href="/wiki/Fuel" title="Fuel">fuels</a> (<a href="/wiki/Methane" title="Methane">methane</a>, <a href="/wiki/Biodiesel" title="Biodiesel">biodiesel</a>, <a href="/w/index.php?title=Biomass&amp;action=edit&amp;redlink=1" class="new" title="Biomass (page disna exeest)">biomass</a>), an <a href="/wiki/Drog" title="Drog">drogs</a> (biopharmaceuticals, <a href="/wiki/Marijuana" class="mw-redirect" title="Marijuana">marijuana</a>, <a href="/wiki/Opium" title="Opium">opium</a>)</p><p>Agricultur micht hae stairtit mair nor 10,000 year syne, but naebody kens for shuir hou auld it is. Agricultur stairtit in the <a href="/w/index.php?title=Fertile_Crescent&amp;action=edit&amp;redlink=1" class="new" title="Fertile Crescent (page disna exeest)">Fertile Crescent</a> in <a href="/wiki/East_Asie" title="East Asie">East Asie</a>. The airt cried the Fertile Crescent is nou in the kintras o <a href="/wiki/Iraq" title="Iraq">Iraq</a>, <a href="/wiki/Sirie" title="Sirie">Sirie</a>, <a href="/wiki/Turkey" title="Turkey">Turkey</a>, <a href="/wiki/Jordan" title="Jordan">Jordan</a>, <a href="/wiki/Lebanon" title="Lebanon">Lebanon</a>, <a href="/wiki/Israel" title="Israel">Israel</a>, an <a href="/wiki/Egyp" title="Egyp">Egyp</a>. <a href="/wiki/Wheat" title="Wheat">Wheat</a> an <a href="/wiki/Baurley" title="Baurley">baurley</a> is some o the first craps fowk grew. Fowk aiblins stairtit agricultur by plautin a puckle craps, but aye <a href="/w/index.php?title=Hunter-gaither&amp;action=edit&amp;redlink=1" class="new" title="Hunter-gaither (page disna exeest)">gaithert</a> mony fuids frae the wild. Fowk micht hae stairtit fermin acause the <a href="/wiki/Wather" title="Wather">wather</a> an <a href="/wiki/Syle" title="Syle">syle</a> begoud tae chynge. <a href="/wiki/Ferm" title="Ferm">Fermin</a> can feed mony mair fowk nor gaitherin on the same bit grund.</p><p>Mony fowk leeves by tawin awa at whit is cried subseestence agricultur, on a wee ferm. Anerly the fermer's faimily leeves on the ferm. Subseestence agricultur is growin juist eneuch fuid for tae feed the fermer, his faimily, an his beasts. Extrae fuid or ainimals is selt for <a href="/wiki/Siller" title="Siller">siller</a> or ither things the fermer canna growe. The yield is the amoont o fuid grown on a gien bit laund, an the yield is aften law, acause subseestence fermers is less eddicate, an they hae less siller for tae coff graith. Whan yields is law, <a href="/wiki/Forest" title="Forest">forests</a> is whiles cuttit for tae fend new laund for tae growe fuid. This is guid in the short term, but can be ill for the <a href="/wiki/Kintra" class="mw-redirect" title="Kintra">kintra</a> an the laund in the lang term.</p><p>In rich kintras, ferms is aften a hail sicht lairger an the yield on the ferms haes gotten bigger in the last hunder year, acause fermers growes better varieties o plaunts, uise mair <a href="/w/index.php?title=Fertilizer&amp;action=edit&amp;redlink=1" class="new" title="Fertilizer (page disna exeest)">fertilizer</a>, uise mair <a href="/wiki/Watter" title="Watter">watter</a>, an haes less <a href="/wiki/Weed" title="Weed">weeds</a> an <a href="/w/index.php?title=Pest&amp;action=edit&amp;redlink=1" class="new" title="Pest (page disna exeest)">pests</a>. Mair oot ower, mony ferms uises machines, sae they need fewer fermers. Sae, there are fewer fermers in rich kintras, but they growe mair fuid. This kind o intensive agricultur maks problems an aw. Fermers uises a lot o chemical fertilizers, <a href="/wiki/Pesticide" title="Pesticide">pesticides</a> (chemicals that kills <a href="/wiki/Insect" title="Insect">insects</a>), an herbicides (chemicals that kills weeds). They whiles pollute the syle or the watter. They whiles mair reseestant insects an weeds. The syle is whiles skaithed by <a href="/wiki/Erosion" title="Erosion">erosion</a>, <a href="/wiki/Saut" title="Saut">saut</a> accumulation, or loss o structur. Irrigation (taen watter frae rivers) can pollute watter an lawer the watter table. Haein fewer fermers chynges society an aw an can mak a kintra less able tae feed itsel in haurd times.</p><table role="presentation" class="mbox-small plainlinks sistersitebox" style="background-color:#f9f9f9;border:1px solid #aaa;color:#000"><tr><td class="mbox-image"><a href="/wiki/File:Commons-logo.svg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Commons-logo.svg/30px-Commons-logo.svg.png" width="30" height="40" class="noviewer" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Commons-logo.svg/45px-Commons-logo.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Commons-logo.svg/59px-Commons-logo.svg.png 2x" data-file-width="1024" data-file-height="1376" /></a></td><td class="mbox-text plainlist">Wikimedia Commons haes media relatit tae <i><b><a href="https://commons.wikimedia.org/wiki/Category:Agriculture" class="extiw" title="commons:Category:Agriculture">Agriculture</a></b></i>.</td></tr></table><!-- NewPP limit reportParsed by mw1337Cached time: 20180307134157Cache expiry: 1900800Dynamic content: falseCPU time usage: 0.064 secondsReal time usage: 0.090 secondsPreprocessor visited node count: 86/1000000Preprocessor generated node count: 0/1500000Post‐expand include size: 2331/2097152 bytesTemplate argument size: 189/2097152 bytesHighest expansion depth: 9/40Expensive parser function count: 0/500Lua time usage: 0.006/10.000 secondsLua memory usage: 571 KB/50 MB--><!--Transclusion expansion time report (%,ms,calls,template)100.00%   60.922      1 Template:Commonscat100.00%   60.922      1 -total 47.13%   28.710      1 Template:Commons 44.40%   27.051      1 Template:Sister 36.23%   22.070      1 Template:Side_box--></div><!-- Saved in parser cache with key scowiki:pcache:idhash:6787-0!canonical and timestamp 20180307134157 and revision id 585337 --><noscript><img src="//sco.wikipedia.org/wiki/Special:CentralAutoLogin/start?type=1x1" alt="" title="" width="1" height="1" style="border: none;
