(Cleveland (USA))</div></div></div><p><b>Pîşesazî</b> an <b>îndustrî</b> an jî <b>endustrî</b> (ji <a href="/wiki/Lat%C3%AEn%C3%AE" class="mw-disambig" title="Latînî">latînî</a> <i>Industria</i>), ji hemû kargeh û karxaneyan, kû bi alîkariya mekîneyan, <a href="/w/index.php?title=Kala&amp;action=edit&amp;redlink=1" class="new" title="Kala (rûpel hê tune)">kala</a>, <a href="/w/index.php?title=Xardemen%C3%AE&amp;action=edit&amp;redlink=1" class="new" title="Xardemenî (rûpel hê tune)">xardemenî</a>, <a href="/w/index.php?title=Ebzar&amp;action=edit&amp;redlink=1" class="new" title="Ebzar (rûpel hê tune)">ebzar</a> yan jî, <a href="/w/index.php?title=Madey%C3%AAn_xav&amp;action=edit&amp;redlink=1" class="new" title="Madeyên xav (rûpel hê tune)">madeyên xav</a> berhev û saz dikin, pêk tê.</p><p>Bi giştî, ji her desteyeka <a href="/wiki/Karwer%C3%AE" title="Karwerî">karweriyê</a> ra ko bi şêwaz û metodên wekhev û hevpişk ji bo bidestveanîna <a href="/w/index.php?title=Fay%C3%AEde&amp;action=edit&amp;redlink=1" class="new" title="Fayîde (rûpel hê tune)">fayîde</a> an <a href="/w/index.php?title=Qezanc&amp;action=edit&amp;redlink=1" class="new" title="Qezanc (rûpel hê tune)">qezancê</a> di xebitin <b>pîşesazî</b> dihêt gotin, mîna <i>pîşesaziya muzîkê</i>, <i>pîşesaziya tirompêlan</i> ango otomobîlan. Bêjeyên hemmehne (sînonîm) endustrî an îndustrî, senayî ne.</p><p>Bi ramaneka dî, pîşesazî ew tişt bû ko wekî bingehê peydebûna <a href="/wiki/Ewropa" title="Ewropa">Ewropa</a> ya nû piştî <a href="/wiki/%C5%9Eore%C5%9Fa_P%C3%AE%C5%9Fesaziy%C3%AA" class="mw-redirect" title="Şoreşa Pîşesaziyê">Şoreşa Pîşesaziyê</a> dihêt hesibandin. Di vê ramana nû da, pîşesaziyê wekî metodekê nû yê berhemanîn û aboriyê cihê berhemanîna <a href="/w/index.php?title=Bazirgan%C3%AE&amp;action=edit&amp;redlink=1" class="new" title="Bazirganî (rûpel hê tune)">bazirganane</a> û <a href="/w/index.php?title=Feudal%C3%AEzm&amp;action=edit&amp;redlink=1" class="new" title="Feudalîzm (rûpel hê tune)">feudalane</a> girt. Destpêka wan guhartinan bi afirandina <a href="/w/index.php?title=Mak%C3%AEneya_helm%C3%AA&amp;action=edit&amp;redlink=1" class="new" title="Makîneya helmê (rûpel hê tune)">makîneya helmê</a> û pêşkevtinên <a href="/wiki/Zanist" title="Zanist">zanistî</a> û <a href="/w/index.php?title=Teknoloc%C3%AE&amp;action=edit&amp;redlink=1" class="new" title="Teknolocî (rûpel hê tune)">teknolocîkî</a> yên dî bû.</p><p>Îndustrî di <a href="/w/index.php?title=Aboriya&amp;action=edit&amp;redlink=1" class="new" title="Aboriya (rûpel hê tune)">aboriya</a> her welatekê, rolekê gelek giring dibîne. <a href="/wiki/Endezyariya_endustr%C3%AE" class="mw-redirect" title="Endezyariya endustrî">Endezyariya endustrî</a>, karê pilan û sazkirina van kargeh û karxaneyan dike û prosêsa avakirinê heta bi Dawî hatin û karkirinê jî kontrol dike.</p><h3><span id="P.C3.AE.C5.9Fesaziya_destp.C3.AAk.C3.AA"></span><span class="mw-headline" id="Pîşesaziya_destpêkê">Pîşesaziya destpêkê</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=P%C3%AE%C5%9Fesaz%C3%AE&amp;action=edit&amp;section=1" title="Beşê biguherîne: Pîşesaziya destpêkê">biguherîne</a><span class="mw-editsection-bracket">]</span></span></h3><p>Ev pişik ji pîşesaziya her neteweyekê wan pîşesaziya vedigirit ko ji bo hemî pîşesaziyên dî gelek bingehîn û destpêkanî ne. Mînakên vî coreyê pîşesaziyê derêxistina madeyên xav ji madenan, cotkarî û çandin, rêlvanî (daristan), terşdarî (xwedîkirina pez û ga û lawirên dî), masîgirî û hwd ne.</p><h2><span id="Curey.C3.AAn_p.C3.AE.C5.9Fesaziy.C3.AA"></span><span class="mw-headline" id="Cureyên_pîşesaziyê">Cureyên pîşesaziyê</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=P%C3%AE%C5%9Fesaz%C3%AE&amp;action=edit&amp;section=2" title="Beşê biguherîne: Cureyên pîşesaziyê">biguherîne</a><span class="mw-editsection-bracket">]</span></span></h2><p>Pîşesazî di <a href="/wiki/Abor%C3%AE" title="Aborî">aboriyê</a> da dabêşî <a href="/w/index.php?title=P%C3%AE%C5%9Fesaziya_destp%C3%AAk&amp;action=edit&amp;redlink=1" class="new" title="Pîşesaziya destpêk (rûpel hê tune)">pîşesaziya destpêk</a>, <a href="/w/index.php?title=P%C3%AE%C5%9Fesaziya_duy%C3%AA&amp;action=edit&amp;redlink=1" class="new" title="Pîşesaziya duyê (rûpel hê tune)">pîşesaziya duyê</a> û <a href="/w/index.php?title=P%C3%AE%C5%9Fesaziya_s%C3%AAy%C3%AA&amp;action=edit&amp;redlink=1" class="new" title="Pîşesaziya sêyê (rûpel hê tune)">pîşesaziya sêyê</a> dibêt. Pîşesaziya duyê bi xwe dabêşî <a href="/w/index.php?title=P%C3%AE%C5%9Fesaziya_sivik&amp;action=edit&amp;redlink=1" class="new" title="Pîşesaziya sivik (rûpel hê tune)">pîşesaziya sivik</a> û <a href="/w/index.php?title=P%C3%AE%C5%9Fesaziya_giran&amp;action=edit&amp;redlink=1" class="new" title="Pîşesaziya giran (rûpel hê tune)">pîşesaziya giran</a> dibit.</p><h2><span id=".C5.9Eaxey.C3.AAn_.C3.8Endustr.C3.AE"></span><span class="mw-headline" id="Şaxeyên_Îndustrî">Şaxeyên Îndustrî</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=P%C3%AE%C5%9Fesaz%C3%AE&amp;action=edit&amp;section=3" title="Beşê biguherîne: Şaxeyên Îndustrî">biguherîne</a><span class="mw-editsection-bracket">]</span></span></h2><p>Îndustrî ji gelek Şaxe yan ji Beşan pêk tê.
