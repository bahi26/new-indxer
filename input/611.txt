Aprika</a></div></div></div><p>Ang <b>agrikultura</b> ay ang paglinang at pagpaparami ng mga <a href="/wiki/Hayop" title="Hayop">hayop</a>, <a href="/wiki/Halaman" title="Halaman">halaman</a> at <a href="/wiki/Halamang-singaw" class="mw-redirect" title="Halamang-singaw">halamang-singaw</a> para gawing <a href="/wiki/Pagkain" title="Pagkain">pagkain</a>, hibla, panggatong, gamot at iba pang mga produkto para gamitin sa pagpapanatili at mapabuti ang <a href="/wiki/Buhay" title="Buhay">buhay</a> ng mga <a href="/wiki/Tao" title="Tao">tao</a>.<sup id="cite_ref-Office1999_1-0" class="reference"><a href="#cite_note-Office1999-1">[1]</a></sup> Naging susi ang agrikultura sa pagsulong ng <a href="/wiki/Kabihasnan" title="Kabihasnan">kabihasnan</a> na nagdolot sa pamumuhay na nakahimpil lamang o <i>sedentary</i>. Nangyari ang ganoong uri ng pamumuhay dahil ang pagpapaamo o pagdomestikado ng mga <a href="/wiki/Espesye" title="Espesye">espesye</a> ay nakalikha ng mga kalabisan sa pagkain. Tinatawag na <a href="/w/index.php?title=Agham_pang-agrikultura&amp;action=edit&amp;redlink=1" class="new" title="Agham pang-agrikultura (hindi pa naisusulat)">agham pang-agrikultura</a> ang pag-aaral agrikultura. Nasa libong taon ang <a href="/wiki/Kasaysayan" title="Kasaysayan">kasaysayan</a> ng agrikultura at ang pag-unlad nito ay lubhang pinapatakbo at binibigay kahulugan ng iba't ibang mga <a href="/wiki/Klima" title="Klima">klima</a>, <a href="/wiki/Kalinangan" title="Kalinangan">kalinangan</a>, at <a href="/wiki/Teknolohiya" title="Teknolohiya">teknolohiya</a>. Namamayani bilang kaparaanang pang-agrikultura ang agrikulturang pang-industriya na nakabatay sa malawakang monokulturang pagsasaka.</p><p>Maaring magkasingkahulugan ang katawagang <a href="/wiki/Pagsasaka" class="mw-redirect" title="Pagsasaka">pagsasaka</a> sa agrikultura ngunit sa mas teknikal na kahulugan, ang pagsasaka ay ang aktibidad samantala ang agrikultura ay ang agham ng pagsasaka at ibang katulad na aktibidad.</p><h2><span class="mw-headline" id="Mga_sanggunian">Mga sanggunian</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Agrikultura&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Baguhin seksiyon: Mga sanggunian">baguhin</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Agrikultura&amp;action=edit&amp;section=1" title="Baguhin seksiyon: Mga sanggunian">baguhin ang batayan</a><span class="mw-editsection-bracket">]</span></span></h2><div class="reflist" style="list-style-type: decimal;"><ol class="references"><li id="cite_note-Office1999-1"><span class="mw-cite-backlink"><a href="#cite_ref-Office1999_1-0">↑</a></span> <span class="reference-text"><span class="citation book">(sa Ingles) <a rel="nofollow" class="external text" href="https://books.google.com/books?id=GtBa6XIW_aQC&amp;pg=PA77"><i>Safety and health in agriculture</i></a>.
