Därutöver finns mer detaljerade grenar som <a href="/wiki/R%C3%A5varuindustri" class="mw-redirect" title="Råvaruindustri">råvaruindustri</a>, <a href="/wiki/Textilindustri" title="Textilindustri">textilindustri</a>, <a href="/wiki/Byggindustri" title="Byggindustri">byggindustri</a>, <a href="/wiki/Bilindustri" class="mw-redirect" title="Bilindustri">bilindustri</a> m.fl.</p><p>I vidare benämning är industri all form av storskaligt <a href="/wiki/N%C3%A4ringsliv" title="Näringsliv">näringsliv</a> - som exempel finns begreppen <a href="/w/index.php?title=Turistindustri&amp;action=edit&amp;redlink=1" class="new" title="Turistindustri [inte skriven än]">turistindustri</a>, <a href="/wiki/N%C3%B6jesindustri" title="Nöjesindustri">nöjesindustri</a> och <a href="/wiki/Filmindustri" class="mw-redirect" title="Filmindustri">filmindustri</a>. Den vidare betydelsen har sitt ursprung i påverkan från <a href="/wiki/Engelska" title="Engelska">engelska</a>, där ordet <i>industry</i> har en bredare innebörd än vad ordet <i>industri</i> har haft i svenska.</p><div id="toc" class="toc"><div class="toctitle" lang="sv" dir="ltr"><h2>Innehåll</h2></div><ul><li class="toclevel-1 tocsection-1"><a href="#Historik"><span class="tocnumber">1</span> <span class="toctext">Historik</span></a><ul><li class="toclevel-2 tocsection-2"><a href="#Industriella_revolutionen"><span class="tocnumber">1.1</span> <span class="toctext">Industriella revolutionen</span></a></li><li class="toclevel-2 tocsection-3"><a href="#Andra_industriella_revolutionen"><span class="tocnumber">1.2</span> <span class="toctext">Andra industriella revolutionen</span></a></li></ul></li><li class="toclevel-1 tocsection-4"><a href="#Typer_av_industri"><span class="tocnumber">2</span> <span class="toctext">Typer av industri</span></a></li><li class="toclevel-1 tocsection-5"><a href="#Se_även"><span class="tocnumber">3</span> <span class="toctext">Se även</span></a></li><li class="toclevel-1 tocsection-6"><a href="#Externa_länkar"><span class="tocnumber">4</span> <span class="toctext">Externa länkar</span></a></li></ul></div><h2><span class="mw-headline" id="Historik">Historik</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Industri&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Redigera avsnitt: Historik">redigera</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Industri&amp;action=edit&amp;section=1" title="Redigera avsnitt: Historik">redigera wikitext</a><span class="mw-editsection-bracket">]</span></span></h2><p>Människor har tillverkat föremål sedan <a href="/wiki/F%C3%B6rhistorisk_tid" title="Förhistorisk tid">förhistorisk tid</a>, men under årtusendena har de framför allt gjort det för det egna hushållet - se <a href="/wiki/Sj%C3%A4lvhush%C3%A5ll_(ekonomiskt_system)" class="mw-redirect" title="Självhushåll (ekonomiskt system)">självhushåll</a>. Under <a href="/wiki/Neolitiska_revolutionen" class="mw-redirect" title="Neolitiska revolutionen">neolitiska revolutionen</a> uppstod <a href="/wiki/By" title="By">byar</a> och senare <a href="/wiki/St%C3%A4der" class="mw-redirect" title="Städer">städer</a>, där en <a href="/wiki/Samh%C3%A4llsklass" title="Samhällsklass">samhällsklass</a> av professionella <a href="/wiki/Hantverkare" title="Hantverkare">hantverkare</a> blev den första industrin. Men det var först på 1700-talet som industrin blev en central del av samhället.</p><p>Ordet industri kommer av latinets <i>industria</i>, som betyder flit, idoghet, sysselsättning. I äldre svenska har också ordet "<a href="/wiki/Sl%C3%B6jd" title="Slöjd">slöjd</a>" använts i betydelsen tillverkning, liksom "<a href="/wiki/Manufaktur" title="Manufaktur">manufaktur</a>" som kommer av latinets ord för <a href="/wiki/Hantverk" title="Hantverk">hantverk</a>. Ett känt exempel är att <a href="/wiki/Chalmers_tekniska_h%C3%B6gskola" title="Chalmers tekniska högskola">Chalmers tekniska högskola</a> under 1800-talet bar namnet "slöjdinstitut". Dagens fabriker har vuxit fram ur det småskaliga hantverket. Pådrivande faktorer för storskaligheten har varit svårtillgängliga råvaruresurser som gruvor och tillgången på mekanisk kraft genom vattenhjul, väderkvarnar och <a href="/wiki/%C3%85ngmaskin" title="Ångmaskin">ångmaskiner</a>. När det blev aktuellt att skilja mellan storskalig industri och småskaligt hantverk, infördes beteckningen hemslöjd eller husflit för det senare.</p><h3><span class="mw-headline" industriella revolutionen kallas de omfattande <a href="/wiki/Samh%C3%A4lle_(sociologi)" title="Samhälle (sociologi)">sociala</a>, <a href="/wiki/Ekonomi" title="Ekonomi">ekonomiska</a> och <a Det är idag ett av Storbritanniens <a href="/wiki/V%C3%A4rldsarv" title="Världsarv">världsarv</a>. </p><p>Den industriella revolutionens framsteg fick allt större betydelse när ångdrivna <a href="/wiki/%C3%85ngfartyg" title="Ångfartyg">fartyg</a> och <a href="/wiki/J%C3%A4rnv%C3%A4g" title="Järnväg">järnvägar</a> blev vanligare. Under <a href="/wiki/1800-talet" title="1800-talet">1800-talet</a> spreds omvälvningen till <a href="/wiki/V%C3%A4steuropa" title="Västeuropa">Västeuropa</a>, <a href="/wiki/Nordamerika" title="Nordamerika">Nordamerika</a> och <a href="/wiki/Japan" title="Japan">Japan</a> och senare till resten av världen. Konsekvensen blev omfattande förflyttningar av befolkningen, från <a href="/wiki/Landsbygd" title="Landsbygd">landsbygden</a> till <a href="/wiki/Stad" title="Stad">städernas</a> <a href="/wiki/Industrisamh%C3%A4lle" title="Industrisamhälle">industrisamhällen</a> som byggdes upp (se <a href="/wiki/Urbanisering" title="Urbanisering">urbanisering</a>).</p><h3><span class="mw-headline" id="Andra_industriella_revolutionen">Andra industriella revolutionen</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Industri&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Redigera avsnitt: Andra industriella revolutionen">redigera</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Industri&amp;action=edit&amp;section=3" title="Redigera avsnitt: Andra industriella revolutionen">redigera wikitext</a><span class="mw-editsection-bracket">]</span></span></h3><div class="noprint huvudartikel"><dl><dd><i>Huvudartikel&#58; <a href="/wiki/Andra_industriella_revolutionen" title="Andra industriella revolutionen">Andra industriella revolutionen</a></i></dd></dl></div><p><b>Andra industriella revolutionen</b> kallas ibland tiden <a href="/wiki/1871" title="1871">1871</a>-<a href="/wiki/1914" title="1914">1914</a>, som bland annat präglades av utbredningen av <a href="/wiki/Elektricitet" title="Elektricitet">elektricitet</a> och <a href="/wiki/F%C3%B6rbr%C3%A4nningsmotor" title="Förbränningsmotor">förbränningsmotorer</a>. Åren som valts att använda för denna tid är den mellan <a href="/wiki/Fransk-tyska_kriget_1870-71" class="mw-redirect" title="Fransk-tyska kriget 1870-71">fransk-tyska kriget</a> <a href="/wiki/1871" title="1871">1871</a> och <a href="/wiki/F%C3%B6rsta_v%C3%A4rldskriget" title="Första världskriget">första världskriget</a>, som bröt ut <a href="/wiki/1914" title="1914">1914</a> och slutade <a href="/wiki/1918" title="1918">1918</a>.</p><h2><span class="mw-headline" id="Typer_av_industri">Typer av industri</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Industri&amp;veaction=edit&amp;section=4" class="mw-editsection-visualeditor" title="Redigera avsnitt: Typer av industri">redigera</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Industri&amp;action=edit&amp;section=4" title="Redigera avsnitt: Typer av industri">redigera wikitext</a><span class="mw-editsection-bracket">]</span></span></h2><p>Det finns många sätt att klassificera industrier. Vanligen avses, som ovan, <a href="/wiki/Tillverkningsindustri" title="Tillverkningsindustri">tillverkningsindustri</a>, där råvaror förädlas till produkter, till skillnad från <a href="/wiki/Tj%C3%A4nsteproduktion" title="Tjänsteproduktion">tjänsteproduktion</a>.</p><p><a href="/wiki/Tung_industri" title="Tung industri">Tung industri</a> är ett begrepp med olika betydelser; rent allmänt syftar det på industri med tunga <a href="/wiki/Insatsvaror" class="mw-redirect" title="Insatsvaror">insatsvaror</a> och produkter; särskilt metallvaror. <b>Basindustri</b> är ett snarlikt begrepp.</p><p>I <a href="/wiki/Processindustri" class="mw-redirect" title="Processindustri">processindustri</a> tillverkas produkter, halvfabrikat och material genom <a href="/w/index.php?title=Fysikalisk_process&amp;action=edit&amp;redlink=1" class="new" title="Fysikalisk process [inte skriven än]">fysikaliska</a> och <a href="/wiki/Kemisk_process" title="Kemisk process">kemiska processer</a>. Dit räknas bland annat <a href="/wiki/Metallindustri" class="mw-redirect" title="Metallindustri">metallindustri</a>, <a href="/wiki/Pappersindustri" class="mw-redirect" title="Pappersindustri">pappersindustri</a> och <a href="/wiki/Kemisk_industri" title="Kemisk industri">kemisk industri</a>.</p><p>I <a href="/wiki/Verkstadsindustri" class="mw-redirect" title="Verkstadsindustri">verkstadsindustri</a> sammansätts <a href="/wiki/Maskin" title="Maskin">maskiner</a> och andra produkter som genom <a href="/wiki/Montering" title="Montering">montering</a>.</p><h2><span id="Se_.C3.A4ven"></span><span class="mw-headline" id="Se_även">Se även</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a
