Kuvassa <a href="/wiki/Altia" title="Altia">Altian</a> tehdas <a href="/wiki/Ilmajoki" title="Ilmajoki">Ilmajoella</a>.</div></div></div><p><b>Teollisuus</b> (lyhenne teoll.<sup id="cite_ref-kotus_1-0" class="reference"><a href="#cite_note-kotus-1">&#91;1&#93;</a></sup>) on <a href="/wiki/Tuotantotekniikka" title="Tuotantotekniikka">tuotantotoiminnan</a> laji, joka jalostaa <a href="/wiki/Luonnonvara" class="mw-redirect" title="Luonnonvara">luonnonvaroja</a> <a href="/wiki/Tuote" title="Tuote">tuotteiksi</a> eli <a href="/wiki/Raaka-aine" title="Raaka-aine">raaka-aineita</a> ihmiselle aineellista hyötyä tuottavaan muotoon. Teollisuus hahmotetaan ns. sekundaarituotannoksi, joka jalostaa erilaisiksi tuotteiksi primäärituotannon eli <a href="/wiki/Alkutuotanto" title="Alkutuotanto">alkutuotannon</a> luovuttamia jalostuskelpoisia <a href="/wiki/Luonnonvara" class="mw-redirect" title="Luonnonvara">luonnonvaroja</a>. Pääasiallisesti teollisuudessa käytetään <a href="/wiki/S%C3%A4hk%C3%B6" title="Sähkö">sähkötoimisia</a> <a href="/wiki/Kone" title="Kone">koneita</a> erotuksena pääasiallisesti ihmisvoimaan perustuvaan <a href="/wiki/K%C3%A4sity%C3%B6" title="Käsityö">käsityöhön</a>. Teollinen valmistus tapahtuu kiinteässä toimipaikassa sijaitsevassa <a href="/wiki/Tehdas" title="Tehdas">tehtaassa</a>, joka on tuotteen valmistamiseen tarvittavaan <a href="/wiki/Tuotantotekniikka" title="Tuotantotekniikka">tuotantotekniikkaan</a> erikoistunut tuotantolaitos.</p><p>Teollisuus on vahvasti sidoksissa <a href="/wiki/Kemia" title="Kemia">kemian</a> ja <a href="/wiki/Fysiikka" title="Fysiikka">fysiikan</a> tieteenaloihin ja teollisuutta varten ovat kehittyneet oppiaineet <a href="/wiki/Tuotantotalous" title="Tuotantotalous">tuotantotalous</a> ja <a href="/wiki/Tuotantotekniikka" title="Tuotantotekniikka">tuotantotekniikka</a>, ja teollistuminen antoi syyn perustaa <a href="/wiki/Teknillinen_korkeakoulu" title="Teknillinen korkeakoulu">teknillisiä korkeakouluja</a>, <a href="/wiki/Kauppakorkeakoulu" title="Kauppakorkeakoulu">kauppakorkeakouluja</a> ja <a href="/wiki/Taideteollinen_korkeakoulu" class="mw-redirect" title="Taideteollinen korkeakoulu">taideteollisia korkeakouluja</a>. Teollisuus sai alkunsa <a href="/wiki/Teollinen_vallankumous" title="Teollinen vallankumous">teollisessa vallankumouksessa</a> 1700-luvun <a href="/wiki/Brittil%C3%A4inen_imperiumi" title="Brittiläinen imperiumi">Brittiläisen imperiumin</a> <a href="/wiki/Englanti" title="Englanti">Englannissa</a>, joka on nykyään osa <a href="/wiki/Yhdistynyt_kuningaskunta" title="Yhdistynyt kuningaskunta">Yhdistynyttä kuningaskuntaa</a>, ja levisi muualle Eurooppaan ja Pohjois-Amerikkaan sekä vähitellen kaikkialle maailmaan. Tämä teollistuminen on vieläkin käynnissä, maailman teollisuustuotannolla on kasvutrendi ja teollistuminen synnyttää <a href="/wiki/Teollisuusyhteiskunta" title="Teollisuusyhteiskunta">teollisuusyhteiskuntia</a> ja <a href="/wiki/Teollisuusmaa" title="Teollisuusmaa">teollisuusmaita</a>, erotuksena teollistumattomista <a href="/wiki/Kehitysmaa" title="Kehitysmaa">kehitysmaista</a>.</p><div id="toc" class="toc"><div class="toctitle" lang="fi" dir="ltr"><h2>Sisällysluettelo</h2></div><ul><li class="toclevel-1 tocsection-1"><a class="tocnumber">6</span> <span class="toctext">Aiheesta muualla</span></a></li></ul></div><h2><span class="mw-headline" id="Teollisuuden_edellytykset_ja_sijoittuminen">Teollisuuden edellytykset ja sijoittuminen</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Teollisuus&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Muokkaa osiota Teollisuuden edellytykset ja sijoittuminen">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Teollisuus&amp;action=edit&amp;section=1" title="Muokkaa osiota Teollisuuden edellytykset ja sijoittuminen">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h2><p>Teollisuuden edellytykset ovat teollisuustuotannossa tarvittavat tuotannontekijät: <a href="/wiki/P%C3%A4%C3%A4oma" class="mw-redirect" title="Pääoma">pääoma</a> (tuotantovälineet; esim. tehdasrakennus, koneet), <a href="/wiki/Raaka-aine" title="Raaka-aine">raaka-aineet</a> (jalostuskelpoiset luonnonvarat joita on tarkoitus jalostaa teollisesti edelleen), <a href="/wiki/Energia" title="Energia">energia</a> (koneiden käyttövoimaksi), <a href="/wiki/Ty%C3%B6ntekij%C3%A4" title="Työntekijä">työntekijät</a> (koneiden käyttäjiksi), <a href="/wiki/Markkinat_(taloustiede)" title="Markkinat (taloustiede)">markkinat</a> (tuotteelle on kysyntää) ja <a href="/wiki/Liikenne" title="Liikenne">liikenneyhteydet</a> (esim. tiet fyysisten tuotannontekijöiden kuljettamiseksi tehtaaseen ja valmiiden tuotteiden kuljettamiseksi ostajille).</p><p>Teollisuustuotannossa tarvittavat tuotantovälineet ovat tuotantohyödykkeitä, jotka on valmistanut joku muu ja joista täytyy maksaa hankkimisvaiheessa. Tuotantovälineiden hankkimista kutsutaan <a href="/wiki/Investointi" title="Investointi">investoimiseksi</a>. Jotta suuria investointeja voitaisiin tehdä, tarvitaan toimivat <a href="/wiki/Rahoitusmarkkinat" title="Rahoitusmarkkinat">rahoitusmarkkinat</a> investointiin tarvittavan rahan keräämiseksi. Rahoitusmarkkinat edellyttävät lainsäädäntöä ja luotettavaa oikeuslaitosta.</p><p>Tärkeitä ovat selkeät ja oikeudenmukaiset yritystoimintaa säätelevät lait, yksityisomistuksen suojaavat lait ja toimivat <a href="/wiki/Yhti%C3%B6muoto" title="Yhtiömuoto">yhtiömuodot</a>. Tärkeitä ovat myös vakaa valuutta, alhainen inflaatio ja alhaiset korot. Jos korot ovat korkeat, vain harvat pitkäaikaiset investointihankkeet ovat kannattavia. Myös vakaat poliittiset olot ovat tärkeitä, jotta tuotantoon ei tule odottamattomia keskeytyksiä. Olemassa ovat muun muassa vallankaappausten, vallankumousten, sotien sekä yritysten kansallistamisten riskit.</p><p>Teollisuus sijoittuu taloudellisen kannattavuuden mukaisesti sinne, missä kokonaiskustannukset ovat matalimmat, koska teollisuus pyrkii kustannustehokkuuteen ja voiton maksimointiin. Osaamisvaltainen teollisuus sijoittuu sinne missä on saatavilla valmiiksi koulutettua ja osaavaa työvoimaa. Työvoimavaltainen teollisuus sijoittuu sinne missä on edullista työvoimaa, esimerkiksi vaatetehtaat Kiinaan. Raaka-ainehakuinen teollisuus sijoittuu sinne mistä saadaan raaka-ainetta, esimerkiksi sellutehtaat. Energiaintensiivinen teollisuus sijoittuu sinne missä on halpaa energiaa, esimerkiksi alumiinisulatot sinne missä on vesivoimalla tuotettua sähköä. Teollisuuden syntyyn tietylle alueelle vaikuttavat myös varsinkin teollisuusmaissa löyhät ympäristösäädökset.</p><h3><span class="mw-headline" id="Raaka-aineet_ja_energia">Raaka-aineet ja energia</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Teollisuus&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Muokkaa osiota Raaka-aineet ja energia">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Teollisuus&amp;action=edit&amp;section=2" title="Muokkaa osiota Raaka-aineet ja energia">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p>Varsinkin teollistumisen alkuaikoina raaka-aineet ja energia olivat teollisuudelle ylivoimaisesti tärkein yksittäinen edellytys, joten teollisuutta syntyi sinne, missä raaka-aineita ja energiaa oli saatavilla. Nykyaikana raaka-ainehakuinen teollisuus käyttää merkittävästi vain vähän jalostettuja raaka-aineita, kuten <a href="/wiki/Maatalous" title="Maatalous">maatalouden</a> tuotteita tai <a href="/wiki/Malmi" title="Malmi">malmeja</a>. Erityisesti rauta- ja terästeollisuus ovat raaka-ainehakuista teollisuutta.</p><p>Teollistuminen alkoi sieltä, missä oli energiaa, pääasiassa <a href="/wiki/Kivihiili" title="Kivihiili">kivihiiltä</a>. Myöhemmin sähkön löytämisen ja <a href="/wiki/S%C3%A4hk%C3%B6generaattori" class="mw-redirect" title="Sähkögeneraattori">sähkögeneraattorin</a> keksimisen jälkeen ryhdyttiin käyttämään kivihiiltä energianlähteenään käyttäneiden höyrykoneiden lisäksi myös sähköä tuottavaa <a href="/wiki/Vesivoima" title="Vesivoima">vesivoimaa</a> – aikaisemmin vesivoimaa oli hyödynnetty mekaanisesti <a href="/wiki/Vesimylly" title="Vesimylly">vesimyllyillä</a>. Ensi kerran sähkön tuotanto aloitettiin Tampereella Tammerkoskessa 22. syyskuuta 1891.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup> Tämä oli Suomen ensimmäinen vesivoimalaitos. Sellu- ja paperiteollisuus on esimerkiksi <a href="/wiki/Suomi" title="Suomi">Suomessa</a> koskien läheisyydessä, ja se tarvitsee paljon vettä valmistusprosessissa. Puut myös historiallisesti uitettiin tehtaaseen vesistöjä pitkin. Vuonna 1928 valmistuneen <a href="/wiki/Imatran_voimalaitos" title="Imatran voimalaitos">Imatran voimalaitoksen</a> perustaminen antoi Suomessa syyn rakentaa valtakunnallinen sähköverkko, joten teollisuuden ei tarvinnut enää sijaita voimalaitoksen välittömässä läheisyydessä.</p><h3><span class="mw-headline" id="Liikenneyhteydet">Liikenneyhteydet</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Teollisuus&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Muokkaa osiota Liikenneyhteydet">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Teollisuus&amp;action=edit&amp;section=3" title="Muokkaa osiota Liikenneyhteydet">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p>Liikenneyhteydet ovat teollisuudelle erittäin tärkeitä, sillä raaka-aineet täytyy kuljettaa tehtaaseen ja valmiit tuotteet markkinoille. Teollisuus sijoittuu tavallisesti hyvien liikenneyhteyksien eli joko vesi-, raide- tai tieyhteyksien tai samaan aikaan näistä useammankin äärelle. Vesiväylät ovat ylivoimaisesti kaikkein energiatehokkain tapa kuljettaa tavaraa, tavarajunat ovat jopa kymmenen kertaa energiatehokkaampia kuin tiekuljetukset.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup></p><p>1800-luvun alussa merten, jokien, järvien ja kanavien vesiyhteydet olivat käytännössä ainoa tapa kuljettaa suuria määriä tavaraa, sillä vaihtoehtoisena kuljetustapana olivat hevosvetoiset kärryt. Vesikuljetukset kehittyivät merten purjealuksista ja kanavien hevosvetoisista <a href="/wiki/Proomu" title="Proomu">proomuista</a> höyrykäyttöiseksi 1800-luvun aikana. Raaka-aineet tavallisesti kuljetettiin kaupungeissa sijaitseviin tehtaisiin, joista tuotteiden kuljetusmatkat kauppoihin olivat lyhyet.</p><p>Myöhemmin 1800-luvulla rakennettiin raideyhteyksiä, jotka mahdollistivat <a href="/wiki/H%C3%B6yryveturi" title="Höyryveturi">höyryveturin</a> ansiosta suurten tavaramäärien, erityisesti raaka-aineiden, kuljettamisen maateitse. Sisämaasta rakennettiin raideyhteyksiä satamiin.</p><p>Moottoriajoneuvot yleistyivät 1900-luvulla ja mahdollistivat niin reitin kuin aikataulunkin suhteen joustavat kuljetukset. Aluksi kuljetuksiin käytettävät ajoneuvot olivat pieniä, nykyisten lava- ja pakettiautojen kaltaisia ja niillä kuljetettiin tavaroita vain lyhyitä matkoja. Myöhemmin rekoilla alettiin kulkea satojen ja jopa tuhansien kilometrien matkoja, koska työvoiman kallistumisen seurauksena ovelta ovelle -kuljetukset olivat useita lastauksia ja purkuja halvempia esim. alle tuhannen kilometrin etäisyyksillä.</p><p>Joillekin teollisuudenaloille ensiarvoisen tärkeää on myynti- ja jakelukustannusten vähäisyys, jolloin ne ovat sijoittuneet pääasiassa sinne, missä markkinat ovat, eli asutuskeskuksien läheisyyteen. Esimerkiksi leipomotuotteet vaativat lyhyet kuljetusmatkat.</p><h3><span id="Ty.C3.B6voima"></span><span class="mw-headline" id="Työvoima">Työvoima</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Teollisuus&amp;veaction=edit&amp;section=4" class="mw-editsection-visualeditor" title="Muokkaa osiota Työvoima">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Teollisuus&amp;action=edit&amp;section=4" title="Muokkaa osiota Työvoima">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p>Jotkin teollisuudenalat vaativat halpoja tai erikoiskoulutettuja työntekijöitä, joten ne sijoittuvat sinne, mistä sellaista on parhaiten saatavilla. Tekstiili- ja elektroniikkateollisuus onkin siirtynyt suurissa määrin halvan työvoiman maihin.</p><h3><span id="Ymp.C3.A4rist.C3.B6n.C3.A4k.C3.B6kulmat"></span><span class="mw-headline" id="Ympäristönäkökulmat">Ympäristönäkökulmat</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Teollisuus&amp;veaction=edit&amp;section=5" class="mw-editsection-visualeditor" title="Muokkaa osiota Ympäristönäkökulmat">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Teollisuus&amp;action=edit&amp;section=5" title="Muokkaa osiota Ympäristönäkökulmat">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p>Huomattava osa teollisuushankkeista <a href="/wiki/Euroopan_unioni" title="Euroopan unioni">Euroopan unionissa</a> on vaarassa ajautua umpikujaan, sillä ne kohtaavat vastustusta paikallisyhteisöiltä ja kansalaisyhteiskunnan organisaatioilta, jotka ovat huolissaan teollisuus- ja <a href="/wiki/Kaivos" title="Kaivos">kaivostoiminnan</a> vaikutuksista <a href="/wiki/Kansanterveys" title="Kansanterveys">kansanterveyteen</a> ja ympäristöön. <a href="/wiki/Teollisuusj%C3%A4te" title="Teollisuusjäte">Teollisuusjäte</a> ja kaivosjäte muodostaa edelleen haasteen lukuisille jäsenvaltioille, joissa on tai on ollut teollisuuslaitoksia ja kaivoksia. Jätteet saattavat olla paikallisyhteisöille joko uhka tai mahdollisuus. Niistä tulee uhka, jos ne vain hylätään ilman toimenpiteitä ympäristöriskien vähentämiseksi, mutta ne voivat jossain tapauksessa muodostua myös mahdollisuudeksi, mikäli jätteen dumppaus synnyttää toimia metallin tai muiden hyödyllisten sekundaaristen raaka-aineiden talteen ottamiseksi. Metallurgisen teollisuuden jätteissä talteenottoteknologiat ovat kehittyneet, ja nyt on mahdollista arvioida uudelleen vanhasta teollisuustoiminnasta aiheutuvan jätteen tarjoamia mahdollisuuksia ja tehdä tästä alasta <a href="/wiki/Ekologinen_kest%C3%A4vyys" title="Ekologinen kestävyys">ekologisesti</a> tarkoituksenmukaista.<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">&#91;4&#93;</a></sup></p><h2><span class="mw-headline" id="Teollisuusalueet">Teollisuusalueet</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Teollisuus&amp;veaction=edit&amp;section=6" class="mw-editsection-visualeditor" title="Muokkaa osiota Teollisuusalueet">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Teollisuus&amp;action=edit&amp;section=6" title="Muokkaa osiota Teollisuusalueet">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h2><div class="thumb tright"><div class="thumbinner" style="width:352px;"><a href="/wiki/Tiedosto:Industria_mundo.png" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ef/Industria_mundo.png/350px-Industria_mundo.png" width="350" height="162" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ef/Industria_mundo.png/525px-Industria_mundo.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ef/Industria_mundo.png/700px-Industria_mundo.png 2x" data-file-width="1357" data-file-height="628" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Tiedosto:Industria_mundo.png" class="internal" title="Suurenna"></a></div>Maailman merkittävimmät teollisuusalueet.</div></div></div><p>Ajan myötä teollisuuslaitokset ovat muodostaneet alueita, joilla toistensa läheisyydessä sijaitsevat tehtaat hyötyvät toisistaan. Esimerkiksi rauta- ja terästehtaat ovat usein toistensa läheisyydessä, ja niiden läheisyyteen syntyy kevyempää teollisuutta. Tehtaiden työntekijät ovat tarvinneet palveluita, jolloin alue on ajan myötä laajennut teollisuuskeskukseksi.</p><h3><span id="L.C3.A4nsi-_ja_Keski-Eurooppa"></span><span class="mw-headline" id="Länsi-_ja_Keski-Eurooppa">Länsi- ja Keski-Eurooppa</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Teollisuus&amp;veaction=edit&amp;section=7" class="mw-editsection-visualeditor" title="Muokkaa osiota Länsi- ja Keski-Eurooppa">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Teollisuus&amp;action=edit&amp;section=7" title="Muokkaa osiota Länsi- ja Keski-Eurooppa">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p>Teollistuminen alkoi Englannissa <a href="/wiki/1700-luku" title="1700-luku">1700-luvulla</a> ja ajan mittaan se levisi <a href="/wiki/Keski-Eurooppa" title="Keski-Eurooppa">Keski-Eurooppaan</a>, ja myöhemmin muille Euroopan alueille. Vanhat teollisuusalueet ovat tiheimmin asuttuja alueita Euroopassa. Nykyaikana Euroopan teollisuus perustuu korkeaan tekniseen kehitykseen ja koulutettuun työvoimaan, kun taas luonnonvarat alkavat olla vähissä ja niitä joudutaan tuomaan halvalla kehitysmaista. Kulkuyhteydet Euroopassa ovat tiheän rauta- ja maantieverkon ansiosta hyvät ja myös rannikoiden satamiin on lähes joka paikasta melko lyhyt matka. Kanavien myötä joet yhdistyvät vesireittien verkostoksi. <a href="/wiki/Euroopan_unioni" title="Euroopan unioni">Euroopan unionin</a> tuoman työvoiman ja raaka-aineiden vapaan liikkumisen myötä Euroopasta on tullut yhä yhtenäisempi kauppa- ja teollisuusalue.</p><h3><span class="mw-headline" id="Yhdysvallat">Yhdysvallat</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Teollisuus&amp;veaction=edit&amp;section=8" class="mw-editsection-visualeditor" title="Muokkaa osiota Yhdysvallat">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Teollisuus&amp;action=edit&amp;section=8" title="Muokkaa osiota Yhdysvallat">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p>Yhdysvallat on ollut maailman johtava teollisuusvaltio <a href="/wiki/1800-luku" title="1800-luku">1800-luvun</a> loppupuolelta saakka. Aluksi maahan kehittyi raskasta teollisuutta mittavien mineraali- ja energiavarojen pohjalta, mutta <a href="/wiki/1900-luku" title="1900-luku">1900-luvun</a> puolivälin tienoilla painopiste on alkanut enemmän siirtyä korkean tekniikan teollisuuteen. Yhdysvaltain teollisuuden vanhinta ydinaluetta ovat itärannikon tiheään asutut <a href="/wiki/Appalakit" title="Appalakit">Appalakkien</a> hiili- ja <a href="/wiki/Yl%C3%A4j%C3%A4rvi" title="Yläjärvi">Yläjärven</a> rautamalmikenttien seudut. Sittemmin teollisuus on levinnyt rannikolta kattamaan koko <a href="/w/index.php?title=Suurten_j%C3%A4rvien_alue&amp;action=edit&amp;redlink=1" class="new" title="Suurten järvien alue (sivua ei ole)">Suurten järvien alueen</a> myös <a href="/wiki/Kanada" title="Kanada">Kanadan</a> puolella. Alueesta käytetään yleisesti nimitystä <i>Yhdysvaltain ja Kanadan teollisuuskolmio</i>, missä terästeollisuuden ympärille on syntynyt auto- ja muuta teollisuutta.</p><p>Koillisosan teollisuuskolmion merkitystä on viime aikoina yhä enemmän vähentänyt <a href="/wiki/Teksas" class="mw-redirect" title="Teksas">Teksasin</a> ja <a href="/wiki/Kalifornia" title="Kalifornia">Kalifornian</a> <i>Aurinkovyöhyke</i>, minkä laajoille teollisuusalueille syntyy jatkuvasti lisää nykyaikaista teollisuutta. Korkeateknologisten tuotteiden ja autojen osalta kilpailu on viime aikoina varsinkin <a href="/wiki/Japani" title="Japani">Japanin</a> ja muiden Aasian maiden ansiosta koventunut, mutta Yhdysvallat on yhä mineraalivarojensa sekä korkeatasoisen koulutuksensa ja tuotekehittelynsä takia edelleen johtava teollisuusmaa.</p><p>Yhdysvaltoihin on kehittynyt lukuisia korkean tekniikan tehtaiden rypäitä, tunnetuimpana Kaliforniassa sijaitseva <a href="/wiki/Piilaakso" title="Piilaakso">Piilaakso</a>, joka tunnetaan pienelektroniikka- ja puolijohdeteollisuudestaan. <a href="/wiki/Boston" title="Boston">Bostoniin</a> on kehittynyt tietokone- ja elektroniikkalaiteteollisuuden ryväs ja <a href="/wiki/Pohjois-Carolina" title="Pohjois-Carolina">Pohjois-Carolinaan</a> on syntynyt biologisten tuotteiden alue. Korkeateknologian ryppäät ovat syntyneet sinne, missä on <a href="/wiki/Yliopisto" title="Yliopisto">yliopistoja</a> ja pääomaa.</p><h3><span id="Ven.C3.A4j.C3.A4_ja_It.C3.A4-Eurooppa"></span><span class="mw-headline" id="Venäjä_ja_Itä-Eurooppa">Venäjä ja Itä-Eurooppa</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Teollisuus&amp;veaction=edit&amp;section=9" class="mw-editsection-visualeditor" title="Muokkaa osiota Venäjä ja Itä-Eurooppa">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Teollisuus&amp;action=edit&amp;section=9" title="Muokkaa osiota Venäjä ja Itä-Eurooppa">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p>Venäjällä suuret ja merkittävät teollisuusalueet ovat <a href="/wiki/Moskovan_alue" title="Moskovan alue">Moskovan alueella</a>, <a href="/wiki/Ural_(vuoristo)" title="Ural (vuoristo)">Uralin</a> itäosissa sekä <a href="/wiki/Siperian_rata" title="Siperian rata">Siperian radan</a> varrella (esimerkiksi <a href="/wiki/Irkutsk" title="Irkutsk">Irkutsk</a> ja <a href="/wiki/Novosibirsk" title="Novosibirsk">Novosibirsk</a>) rautamalmi- ja hiilialueilla mutta myös strategisesti otollisilla paikoilla. Myös <a href="/wiki/Ukraina" title="Ukraina">Ukrainan</a> teollisuus on keskittynyt rautamalmi- ja hiilialueille. Rauta- ja hiilialueet, jotka välttämättä eivät sijaitse toistensa välittömässä läheisyydessä, muodostavat <i>teollisuuskombinaatin</i>, jossa juna ensiksi tuo hiilialueelta hiiltä rauta-alueelle ja toiseen suuntaan menneessään vie rautaa hiilialueelle. Esimerkiksi <a href="/wiki/Dnipro" title="Dnipro">Dnepropetrovskin</a> kaupunki on syntynyt tähän tapaan. <a href="/wiki/Neuvostoliitto" title="Neuvostoliitto">Neuvostoliiton</a> hajoamista seuranneet talouden ja poliittisen elämän muutokset ovat aiheuttaneet teollisuudelle uusia hankaluuksia pääasiassa energian ja raaka-aineiden saannin vaikeuksien muodossa. Ennen raaka-aine- ja energiavaroja ohjailtiin keskusjohtoisesti ja osatasavallat oli erikoistettu kukin usein vain yhteen tiettyyn tuotealueeseen. Asevarustelun osuus oli huomattavasti kulutushyödykkeiden tuottoa suurempi ja tuotteet olivat usein huonolaatuisia ja ympäristöasiat retuperällä. Venäjän teollisuustuotanto onkin nykyaikana vähentynyt, kun hajasijoittunut teollisuus ei voimakkaan keskusjohtoisuuden purkamisen jälkeen ole kyennyt järjestäytymään kovin nopeasti eikä tehokkaasti. Venäjän teollisuustuotannon kivijalka on pinta-alaltaan laajan maan luonnonvaroja hyödyntävä raskas teollisuus. Lisärajoitteen tuovat huonot liikenneyhteydet.</p><h3><span id="It.C3.A4-_ja_Kaakkois-Aasia"></span><span class="mw-headline" id="Itä-_ja_Kaakkois-Aasia">Itä- ja Kaakkois-Aasia</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Teollisuus&amp;veaction=edit&amp;section=10" class="mw-editsection-visualeditor" title="Muokkaa osiota Itä- ja Kaakkois-Aasia">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Teollisuus&amp;action=edit&amp;section=10" title="Muokkaa osiota Itä- ja Kaakkois-Aasia">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p><a href="/wiki/Toinen_maailmansota" title="Toinen maailmansota">Toisen maailmansodan</a> jälkeen <a href="/wiki/Japani" title="Japani">Japani</a> suojasi kotimarkkinansa tuonnilta suojatulleilla ja nousi parissakymmenessä vuodessa maailman johtavien teollisuusvaltojen joukkoon. Maalla on erittäin vähän omia luonnonvaroja ja sen teollisuus perustuukin työvoiman korkeaan koulutukseen ja työmoraaliin sekä jatkuvaan ja pitkälle kehittyneeseen tutkimukseen ja tuotekehittelyyn, jonka tunnetuimpia tuloksia ovat puolijohteet, televisiot, tietokoneet, kamerat, optiset kuidut sekä teollisuusrobotit. Teollisuuskaupungit sijaitsevat rannikoilla poukamissa satamien läheisyydessä. Suurin keskittymä on <a href="/wiki/Tokio" title="Tokio">Tokion</a> ympäristössä, missä monet yliopistot ja tutkimuskeskukset sijaitsevat.</p><p>Kaakkois-Aasian kasvuedellytykset ovat hyvät. Alue oli jo <a href="/wiki/1400-luku" title="1400-luku">1400-luvulla</a> merkittävä silkin, lääkkeiden ja posliinin toimittaja. Alueella on suuret luonnonvarat etenkin <a href="/wiki/%C3%96ljy" title="Öljy">öljyn</a>, <a href="/wiki/Maakaasu" title="Maakaasu">maakaasun</a>, <a href="/wiki/Tina" title="Tina">tinan</a> ja <a href="/wiki/Kumi" title="Kumi">kumin</a> osalta. Väestö on nuorta ja sitä on paljon, joten työvoimaa ja markkinoita on riittävästi. Alue on kehittynyt nopeasti ja neljän ”taloustiikerin”, <a href="/wiki/Singapore" title="Singapore">Singaporen</a>, <a href="/wiki/Hongkong" title="Hongkong">Hongkongin</a>, <a href="/wiki/Etel%C3%A4-Korea" class="mw-redirect" title="Etelä-Korea">Etelä-Korean</a> sekä <a href="/wiki/Taiwan" title="Taiwan">Taiwanin</a>, talouskasvu oli <a href="/wiki/1980-luku" title="1980-luku">1980-</a> ja <a href="/wiki/1990-luku" title="1990-luku">1990-luvulla</a> ennennäkemättömän nopeaa. <a href="/wiki/Korean_tasavalta" title="Korean tasavalta">Korean tasavalta</a> on seurannut Japanin esimerkkiä. Japanin tavoin maan väestöllä on korkea koulutustaso ja työmoraali ja maalla onkin kansainvälisesti korkea asema vaate-, urheilujalkine--, laiva-, auto-, mikrosiru- ja elektroniikkamarkkinoilla. Taiwanin teollisuus perustuu teknisesti kehittyneeseen markkinatalouteen, eikä paikallinen työvoima ole enää edullista, joten yrittäjät ovat siirtäneet tehtaitaan manner-<a href="/wiki/Kiina" title="Kiina">Kiinan</a>. Maiden väliset heikot ulkopoliittiset suhteet kuitenkin aiheuttavat epävarmuutta. Kaakkois-Aasian maista erityisesti <a href="/wiki/Malesia" title="Malesia">Malesia</a>, <a href="/wiki/Thaimaa" title="Thaimaa">Thaimaa</a> ja <a href="/wiki/Indonesia" title="Indonesia">Indonesia</a> tarjoavat huomattavia määriä halpaa työvoimaa, mikä on houkutellut ulkomaisia yrityksiä ja pääomaa.</p><p><a href="/wiki/Kiina" title="Kiina">Kiinalla</a> on pitkät perinteet <a href="/wiki/Silkki" title="Silkki">silkki-</a> ja <a href="/wiki/Puuvilla" title="Puuvilla">puuvillateollisuuden</a> harjoittamisessa ja <a href="/wiki/1500-luku" title="1500-luku">1500-luvulla</a> se oli taloudellisesti kehittynein maa maailmassa. Sittemmin poliittinen epävakaus ja tehoton kommunismihallinto aiheuttivat talouden taantumisen ja vasta <a href="/wiki/1980-luku" title="1980-luku">1980-luvulla</a>, kun talousjärjestelmään tehtiin markkinatalouteen suuntaavia muutoksia, tärkeimpänä avautuminen ulkomaankaupalle. Kiinan luonnonvarat ovat runsaat ja halpaa työvoimaa riittää, mikä on viime aikoina yhä enemmän innostanut ulkomaisia yrityksiä suoriin investointeihin. <a href="/wiki/Intia" title="Intia">Intian</a> kehittyminen on viime aikoina noudattanut pitkälle Kiinan linjoja. Intiassa perinteisiä suuria teollisuusalueita ovat <a href="/wiki/Kalkutta" title="Kalkutta">Kalkutta</a> ja <a href="/wiki/Bombay" class="mw-redirect" title="Bombay">Bombay</a>.</p><h3><span class="mw-headline" id="Latinalainen_Amerikka">Latinalainen Amerikka</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Teollisuus&amp;veaction=edit&amp;section=11" class="mw-editsection-visualeditor" title="Muokkaa osiota Latinalainen Amerikka">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Teollisuus&amp;action=edit&amp;section=11" title="Muokkaa osiota Latinalainen Amerikka">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p>Latinalaisessa Amerikassa on oma teollistuneiden kehitysmaiden ryhmänsä, joka koostuu <a href="/wiki/Argentiina" title="Argentiina">Argentiinasta</a>, <a href="/wiki/Brasilia" title="Brasilia">Brasiliasta</a>, <a href="/wiki/Chile" title="Chile">Chilestä</a> ja <a href="/wiki/Meksiko" title="Meksiko">Meksikosta</a>. Brasilian tärkeimmät teollisuusalueet ovat suurten kaupunkien, <a href="/wiki/S%C3%A3o_Paulo" title="São Paulo">São Paulon</a> ja <a href="/wiki/Rio_de_Janeiro" title="Rio de Janeiro">Rio de Janeiron</a> lähistöllä. <a href="/wiki/Minas_Gerais" title="Minas Gerais">Minas Gerais</a> on maapallon suurimpia <a href="/wiki/Rautamalmi" title="Rautamalmi">rautamalmialueita</a>. Teollisuus perustuu kotimaisiin raaka-aineisiin ja ulottuu monipuolisesti raskaista koneista jokapäiväisen kulutuksen tuotteisiin. Argentiinassa merkittävin teollisuudenala on maataloustuotteiden jalostus, mutta myös kulkuneuvo- ja koneenrakennusteollisuus on maan laajojen kaivannais- ja öljykenttien seurauksena tärkeitä teollisuudenhaaroja. <a href="/wiki/Meksiko" title="Meksiko">Meksikossa</a> on myös paljon malmeja ja öljyä, teollisuus on pitkään ollut monipuolista. Maa on solminut Yhdysvaltojen ja Kanadan kanssa vapaakauppasopimuksen, minkä seurauksena Yhdysvalloista on siirtynyt teollisuutta halvan työvoiman Meksikoon.</p><h3><span id="Etel.C3.A4-Afrikka"></span><span class="mw-headline" id="Etelä-Afrikka">Etelä-Afrikka</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Teollisuus&amp;veaction=edit&amp;section=12" class="mw-editsection-visualeditor" title="Muokkaa osiota Etelä-Afrikka">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Teollisuus&amp;action=edit&amp;section=12" title="Muokkaa osiota Etelä-Afrikka">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p>Afrikassa teollisuutta on lähinnä <a href="/wiki/Etel%C3%A4-Afrikka" title="Etelä-Afrikka">Etelä-Afrikan tasavallassa</a>, joka on maanosan ylivoimaisesti teollistunein maa. Etelä-Afrikan teollisuus on hyvin monipuolista, sillä <a href="/wiki/Apartheid" title="Apartheid">apartheidin</a> aikana maa oli kauppasaarrossa ja joutui näin itse valmistamaan tuotteensa. Nykyään Etelä-Afrikka on aktiivisesti mukana maailmankaupassa ja kansainvälisillä yrityksillä on siellä toimintaa. Maassa on myös laajat kivihiili- ja malmivarat, runsaasti työvoimaa, kasvavat markkinat sekä hyvät liikenneyhteydet ja energiantuotanto.</p><h3><span class="mw-headline" id="Australia">Australia</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Teollisuus&amp;veaction=edit&amp;section=13" class="mw-editsection-visualeditor" title="Muokkaa osiota Australia">muokkaa</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Teollisuus&amp;action=edit&amp;section=13" title="Muokkaa osiota Australia">muokkaa wikitekstiä</a><span class="mw-editsection-bracket">]</span></span></h3><p><a href="/wiki/Australia" title="Australia">Australiassa</a> luonnonvarat ovat valtavat, mutta teollisuustuotanto siihen nähden pientä. Australia vie raaka-aineita ulkomaille: esimerkiksi kivihiiltä ja metalleja. Australian pienen teollisuustuotannon selittää alueen pieni väestömäärä ja raaka-aineiden tarve maapallon suuriväestöisillä alueilla (Pohjois-Amerikka, Eurooppa ja Aasia).</p><h2><span
