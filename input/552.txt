class="magnify"><a href="/wiki/Att%C4%93ls:Arabakolautada.jpg" class="internal" title="Palielināt"></a></div><a href="/w/index.php?title=Aramzeme&amp;action=edit&amp;redlink=1" class="new" title="Aramzeme (vēl nav uzrakstīts)">Aramzeme</a>&#160;— lauksaimniecībā izmantojama zeme</div></div></div><p><b>Lauksaimniecība</b> ir <a href="/wiki/Tautsaimniec%C4%ABba" class="mw-redirect" title="Tautsaimniecība">tautsaimniecības</a> nozare, kas vērsta uz <a href="/wiki/Augi" title="Augi">augu</a> un <a href="/wiki/Dz%C4%ABvnieki" title="Dzīvnieki">dzīvnieku</a> izcelsmes <a href="/wiki/P%C4%81rtika" title="Pārtika">pārtikas</a> produktu iegūšanu un/vai citu to izejvielu iegūšanu <a href="/wiki/R%C5%ABpniec%C4%ABba" title="Rūpniecība">rūpniecības</a> nolūkiem, piemēram, <a href="/wiki/Aitkop%C4%ABba" title="Aitkopība">aitkopji</a> iegūst <a href="/wiki/Vilna" title="Vilna">vilnu</a>, lai pēc tam no tās gatavotu <a href="/wiki/Ap%C4%A3%C4%93rbs" title="Apģērbs">apģērbu</a>. Mūsdienās daudzi lauksaimnieki ir pārorientējušies arī uz <a href="/wiki/Biodegviela" title="Biodegviela">biodegvielas</a> iegūšanu no <a href="/w/index.php?title=Rap%C5%A1i&amp;action=edit&amp;redlink=1" class="new" title="Rapši (vēl nav uzrakstīts)">rapšiem</a>, kas arī ir lauksaimniecības nozare. Zinātni, kas pēta ar lauksaimniecību saistītos jautājumus, sauc par <a href="/wiki/Agronomija" title="Agronomija">agronomiju</a>.</p><p>Lauksaimniecība ir viena no senākajām saimniecības nozarēm, kuras attīstība spēcīgi ietekmējusi cilvēces vēsturi. Aprēķināts, ka ap 40% pasaules iedzīvotāju ir nodarbināti lauksaimniecībā, tomēr kopš <a href="/wiki/R%C5%ABpniecisk%C4%81_revol%C5%ABcija" title="Rūpnieciskā revolūcija">Rūpnieciskās revolūcijas</a> sākuma tās nozīme arvien mazinās, un tā veido tikai 5,7% no pasaules kopprodukta (tas ir no visu pasaules valstu kopējā <a href="/wiki/Iek%C5%A1zemes_kopprodukts" title="Iekšzemes kopprodukts">iekšzemes kopprodukta</a>).<sup id="cite_ref-CIA_1-0" class="reference"><a href="#cite_note-CIA-1">[1]</a></sup></p><p>Lauksaimniecības profesijas var apgūt specializētās skolās, piemēram, <a href="/w/index.php?title=Arodskola&amp;action=edit&amp;redlink=1" class="new" title="Arodskola (vēl nav uzrakstīts)">arodskolās</a>, <a href="/wiki/Koled%C5%BEa" title="Koledža">koledžās</a> un pat <a href="/wiki/Universit%C4%81te" title="Universitāte">universitātēs</a>. <a href="/wiki/Latvija" title="Latvija">Latvijā</a> šāda universitāte ir <a href="/wiki/Latvijas_Lauksaimniec%C4%ABbas_universit%C4%81te" title="Latvijas Lauksaimniecības universitāte">Latvijas Lauksaimniecības universitāte</a> (LLU) <a href="/wiki/Jelgava" title="Jelgava">Jelgavā</a>.</p><p></p><div id="toc" class="toc"><div class="toctitle" lang="lv" dir="ltr"><h2>Satura rādītājs</h2></div><ul><li class="toclevel-1 tocsection-1"><a Arī pirms tam cilvēks izmantoja <a href="/wiki/Flora" title="Flora">floras</a> un <a href="/wiki/Fauna" title="Fauna">faunas</a> sniegtās dabas veltes, tomēr tās tika gūtas, lasot meža veltes vai medījot dzīvniekus, nevis apzināti kultivējot konkrētas augu vai dzīvnieku sugas.
