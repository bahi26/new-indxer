title="Burdingintza">burdingintza</a> sektoreko <b>industria</b>-fabrika handi bat, non burdina ekoizten den.</div></div></div><p><b>Industria</b> <a href="/wiki/Lehengai" title="Lehengai">lehengaiak</a> eraldatuz <a href="/wiki/Produktu_(argipena)" class="mw-disambig" title="Produktu (argipena)">produktuak</a> ekoiztu egiten dituzten jarduera eta prozesuen multzoa da. Ekoiztutako produktuak zein diren, industria-jarduera mota ezberdinak daude. Adibidez, <a href="/wiki/Automobilgintza" title="Automobilgintza">automobilgintza</a> izeneko industria-sektorean, automobilak eta automobiletarako osagaiak ekoizten dituzten jarduerak biltzen dira eta <a href="/wiki/Metalgintza" class="mw-redirect" title="Metalgintza">metalgintzan</a> <a href="/wiki/Burdina" title="Burdina">burdina</a> eta beste <a href="/wiki/Metal" title="Metal">metalak</a> eraldatzen dituzten <a href="/wiki/Fabrika" title="Fabrika">fabrikak</a> biltzen dira<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup>. Lehengaiez gainera, industriak lehengaiak eraldatzeko <a href="/wiki/Energia" title="Energia">energia</a> behar du, baita <a href="/wiki/Azpiegitura" title="Azpiegitura">azpiegiturak</a> ere (<a href="/wiki/Errepide" title="Errepide">errepideak</a>, <a href="/wiki/Trenbide" title="Trenbide">trenbideak</a>, ...) lehengaiak fabriketaraino garraiatu eta bertatik produktu bukatuak banatzeko.</p><p><a href="/wiki/Ekonomia" title="Ekonomia">Ekonomian</a>, industriak <a href="/wiki/Bigarren_sektore" title="Bigarren sektore">bigarren sektore</a> izenekoa osatzen du, <a href="/wiki/Nekazaritza" title="Nekazaritza">nekazaritza</a> eta <a href="/wiki/Arrantza" title="Arrantza">arrantza</a> hartzen duten lehenengo sektorearen eta <a href="/wiki/Zerbitzu_(ekonomia)" title="Zerbitzu (ekonomia)">zerbitzuak</a> barne hartzen dituen hirugarren sektorearen aldean<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup>. <a href="/wiki/Historia" title="Historia">Historian</a> zehar, <a href="/wiki/Industria_Iraultza" title="Industria Iraultza">Industria Iraultza</a> izenekoa izan zen <a href="/wiki/XVIII._mendea" title="XVIII. mendea">XVIII</a> eta <a href="/wiki/XIX._mendea" title="XIX. mendea">XIX. mendeetan</a> zehar industria gehien garatu zen aldia, ekonomia osoaren produktua nabarmen gehituz. Dena den, munduan zehar industriaren garapena oso ezberdina da eta horrela herrialde industrializatuak eta ez-industrializatuak bereizten dira. Aldi berean, herrialde industrializatuak izaten dira ongizate material handiena izaten dutenak, oraindik nekazaritzan oinarritzen diren herrialdeen aldean.</p><p>Hala ere, herrialde aberatsetan badago joera fabrikak kostu txikiagoak dituzten herrialdeetara eraman eta bestelako jarduera espezializatuagoak (hirugarren sektorean, esaterako) sendotzeko. Industriaren garapena izan da baita ere, neurri handi batean, egungo <a href="/wiki/Ingurumen" title="Ingurumen">ingurumen</a>-arazoen sortzaileetako bat, bere ekoizpen-prozesuetan sortutako hondakinak <a href="/wiki/Atmosfera" title="Atmosfera">atmosfera</a> eta <a href="/wiki/Ibai" title="Ibai">ibaietara</a> kontrol eskasarekin isurtzeagatik besteak beste.</p><p></p><div id="toc" class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Industria&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Historia»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Industria&amp;action=edit&amp;section=1" title="Aldatu atal hau: «Historia»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h2><p>Gizakiaren ezaugarria da, animaliak ez bezala, lehengaiak bildu, teknika baliatu eta lan eginez bere beharrak asebetetzeko gauza izatea. Horrela, industria antzinatik erabili du gizakiak. <a href="/wiki/Su" title="Su">Suaren</a> erabilera industria-jardueraren jatorrian dagoela esan daiteke. Gizakiak harriz egindako tresnak ere duela milioi bat urtetik gutxi gorabehera egiten ditu, <a href="/wiki/Paleolito" title="Paleolito">Paleolitiko</a> garaiko <a href="/wiki/Industria_litiko" title="Industria litiko">industria litikoa</a> erakusten duenez. Antzinako gizakiek egurrez eta larruz egindako tresnak ere industria-jardueraren hastapenak direla esan daiteke.</p><p>Industria-jarduera modernoaren ezaugarria den espezializazioa edo lanaren banaketa ere antzinatik gertatzen da. Hasierako espezializazioari ekin zioten langileak <a href="/wiki/Eskulangintza" title="Eskulangintza">artisauak</a> izan ziren, industria jardueran nagusi izan zirenak antzinako <a href="/wiki/Zibilizazio" title="Zibilizazio">zibilizazioetatik</a> <a href="/wiki/XVIII._mendea" title="XVIII. mendea">XVIII. mendean</a> <a href="/wiki/Industria_Iraultza" title="Industria Iraultza">Industria Iraultza</a> abiarazi arte. Artisau bakoitzak produktu-mota zehatz bat ekoizten zuen, berezko edo ikasita zituen teknikak erabiliz eta ekoizpen-prozesu osoa hasieratik bukaerara berak bakarrik burutuz. Bere lanaren xehetasunak sekretupean edukitzen zituen, lanbidearen teknikak aitak eta amak semeari eta alabari bakarrik irakatsiz, industriak ematen zion pribilejioa ez galtzeko. Industria bereko artisau-elkarteetan (zapatariak, errementariak, ...) ere biltzen ziren <a href="/wiki/Erdi_Aroa" title="Erdi Aroa">Erdi Arotik</a> aurrera bere lanbidearen interesak batera hobeto defendatzeko.</p><p><a href="/wiki/XVIII._mendea" title="XVIII. mendea">XVIII. mendean</a> eta <a href="/wiki/Ingalaterra" title="Ingalaterra">Ingalaterran</a> hasita, eta <a href="/wiki/Europa" title="Europa">Europako</a> beste herrialdeetan eta <a href="/wiki/Ameriketako_Estatu_Batuak" title="Ameriketako Estatu Batuak">Ameriketako Estatu Batuetan</a> ondoren, artisau eta merkatari aske eta aberatsek sortutako <a href="/wiki/Gizarte-klase" class="mw-redirect" title="Gizarte-klase">gizarte-klase</a> boteretsu bat sortu zen, <a href="/wiki/Burgesia" title="Burgesia">burgesia</a> izenekoa. Hazkunde demografikoak ekarri zuen langileen ugaritasunak eta, merkataritzaren garapenari esker, lehengaien eskuragarritasunak, industria-produktuak ekoizteko modu berri bat ekarri zuen, nagusi baten mendean langile asko soldata eskasekin fabrika batean bildu eta produktuak askoz ere merkeago ekoizten ziren, lehenengo artisauak ez bezala, ekoizpen-prozesuko faseak zatituz eta makineria berria sartuz. <a href="/wiki/Industria_Iraultza" title="Industria Iraultza">Industria Iraultzaren</a> eta <a href="/wiki/Kapitalismo" title="Kapitalismo">kapitalismoaren</a> sorrera izan zen.</p><div style="font-style: italic; padding-left: 2em; margin-bottom: 0.5em;">Sakontzeko, irakurri: «<a href="/wiki/Industria_Iraultza" title="Industria Iraultza">Industria Iraultza</a>» eta «<a href="/wiki/Bigarren_Industria_Iraultza" title="Bigarren Industria Iraultza">Bigarren Industria Iraultza</a>»</div><h2><span class="mw-headline" id="Industria-motak">Industria-motak</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Industria&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Industria-motak»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Industria&amp;action=edit&amp;section=2" title="Aldatu atal hau: «Industria-motak»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h2><p>Ekoizpen prozesu orokorrean duten kokapenari buruz, industriak honela sailka daitezke:</p><ul><li><b>oinarrizko industriak</b>, lehengaiak eraldatuz produktu erdibukatuak ekoizten dituztenak, beste industria-jarduera batzeutan erabiltzeko; adibidez, <a href="/wiki/Burdingintza" title="Burdingintza">siderurgiak</a> <a href="/wiki/Mea" title="Mea">mineralak</a> eraldatu eta <a href="/wiki/Metal" title="Metal">metal</a> gordina ekoizten du, beste industria batzuetan erabiltzeko;</li><li><b>ekipo-ondasunen industriak</b>, beste jarduera ekonomikoentzako makinak, ibilgailuak eta ekipo elektronikoak ekoizten dituztenak; adibidez, <a href="/wiki/Garabi" title="Garabi">garabiak</a> edo energia eolikoa ekoizteko <a href="/wiki/Errota" title="Errota">errotak</a> ekoizten dituzten fabrikak;</li><li><b>kontsumo-ondasunen industriak</b>, familientzako ondasunak ekoizten dituztenak; hala nola, <a href="/wiki/Ehungintza" title="Ehungintza">ehungintza</a>, etxetresna elektrikoak egiten dituzten fabrikak.</li></ul><p>Erabiltzen duten lehengaien kopuruari buruz, sailkapen hau egin daiteke:</p><ul><li><b>industria astuna</b>, lehengaiak era masiboan erabiltzen dituzten fabrikak; adibidez, <a href="/w/index.php?title=Zementugintza&amp;action=edit&amp;redlink=1" class="new" title="Zementugintza (sortu gabe)">zementugintzan</a> <a href="/wiki/Kareharri" title="Kareharri">kareharri</a> kopuru handiak erabiltzen dira <a href="/wiki/Zementu" title="Zementu">zementua</a> egiteko;</li><li><b>industria erdiarina</b>, produktu erdibukatuekin egiten dute lana baino industria astunek lehengaiekin baino neurri apalagoan; ekipo ondasunen industriak sail honetan kokatzen dira;</li><li><b>industria arina</b>, neurri apalean erabiltzen dituzte lehengaiak edo produktu erdibukatuak; adibidez, ehungintza, <a href="/wiki/Arotzgintza" class="mw-redirect mw-disambig" title="Arotzgintza">arotzgintza</a>.</li></ul><p>Beste alde batetik sailkapen hau ere egin daiteke ekoiztutako gaia zein den:</p><ul><li><b>erauzketa-industriak</b>, lehengaiak naturatik hartzen dituztenak, eraldaketa minimo batekin; adibidez, <a href="/wiki/Meatzaritza" title="Meatzaritza">meatzaritza</a>;</li><li><b>energia-industriak</b>, energia ekoizteari ekiten diotenak;</li><li><b>eraldaketa-industriak</b>, produktu bukatuak edo erdibukatuak ekoizten dituztenak, lehengaiak edo produktu erdibukatuak erabiliz;</li><li><b>muntaia-industriak</b>, osagaiak eta bestelako produktuak osatzen ditiztenak, produktu bukatu bat osatzeko; adibidez, automobilgintza.</li></ul><p>Bere garapenari buruz, industriak puntakoak edo helduak izan daitezke:</p><ul><li><b>puntako industriak</b>, azkar garatzen ari diren industriak, teknologia maila aurreratu batekin; adibidez, aeronautika-industria.</li><li><b>industria helduak</b>, teknologia egonkor batekin, garapen-muga batera heldu direnak; metalurgia esaterako.</li></ul><p>Azkenik, industriak langile kopuruari buruz ere sailka daitezke. Horrela, industria txikiak (langile kopurua 50 baino txikiagoa denean), ertainak (50-100) eta handiak (100 langile baino gehiago) bereizten dira, langile kopuruaren mugak herrialdea zein den aldakorrak badira ere. Aldi berean, lan-indar intentsiboko industria (langile asko behar ditiztenak, kapital finkoaren aldean; <a href="/wiki/Ehungintza" title="Ehungintza">ehungintza</a>, esaterako) eta kapital intensiboko industria (makineria eta azpiegitura handiak behar dituena, lan indarraren aldean; energia-industria, esaterako) bereizten dira.</p><h2><span class="mw-headline" id="Industriaren_kokapena">Industriaren kokapena</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Industria&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Industriaren kokapena»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Industria&amp;action=edit&amp;section=3" title="Aldatu atal hau: «Industriaren kokapena»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h2><p>Industria-jarduerak ez dira uniformeki banatzen lurraldean zehar. Fabrikek bata bestearen ondoan kokatzeko joera izaten dute, beti horrela ez bada ere, azpiegiturak eta elkarrekiko osagarritasuna (aglomerazio-ekonomiak deritzenak) baliatzeko. Kokapen industrialerako kontuan harturiko faktoreak hauek izaten dira: <a href="/wiki/Lehengai" title="Lehengai">lehengaien</a> eskuragarritasuna, industria osagarrien gertutasuna, merkatu potentzialerako irisgarritasuna (urbanizazio-ekonomiak), lan-indarraren kostuak, azpiegiturak eta aglomerazio-ekonomiak <sup id="cite_ref-3" class="reference"><a href="#cite_note-3">[3]</a></sup>. Fiskalitatea (ordaindu beharreko zergen zenbatekoa) eta langileen prestakuntza ere kontuan hartu beharreko faktoreak izaten dira. Industria handien kasuan, faktore <a href="/wiki/Geopolitika" title="Geopolitika">geopolitikoak</a> ere erabakigarriak izan daitezke.</p><p>Kontsumo-ondasunen industrietan, fabrikak hiri handien ondoko <a href="/wiki/Hinterland" title="Hinterland">hinterland</a> izeneko eremuetan lekutu ohi dira, hiri handietako merkatuak eta azpiegitura-sare zabalak direla eta. Oinarrizko industrietan, ordea, lehengai-iturrien gertu izatea faktore erabakigarria izaten da; adibidez, ohizkoa da petrolio-birfindegiak petrolio-putzuetatik gertu kokatu izan dira. Egun ordea, lehengaien faktore ez da lehen bezain erabakigarria izaten; adibidez, petrolioaren, oliobide-sareei esker birfindegiak petrolio-putzuetatik urrun ere kokatzen dira. Ekipo-ondasunetan, garrantzitsua izaten da inguruko fabriken osagarritasuna: adibidez, <a href="/wiki/Automobil" title="Automobil">automobilgintzako</a> fabrikek denetariko piezak eta osagaiak ekoizten dituzten fabrikak behar izaten dituzte euren inguruan. Bertako biztanleen prestakuntza teknologia maila aurreratua duten industrietarako beharrezkoa da. Lan-indarraren kostua, herrialde batetik bestera gehienetan, gero eta gehiago kontuan hartzen den faktorea da, <a href="/wiki/Globalizazio" title="Globalizazio">globalizazioa</a> dela eta, produktuak konkurrentziak baino merkeago eskaintzeko presioa oso handia izaten baita. Horrela, <a href="/wiki/Deslokalizazio" title="Deslokalizazio">deslokalizazio</a> fenomenoa gertatzen da askotan, non bertako fabrikak itxi egiten diren, soldata merkeagoak dituzten herrialdeetan zabaltzeko. Bertako kultura industriala, antolaketa-modua eta babes politikoa ere kontuan hartzen diren faktoreak dira.</p><h2><span class="mw-headline" id="Paisaia_industrialak">Paisaia industrialak</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Industria&amp;veaction=edit&amp;section=4" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Paisaia industrialak»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Industria&amp;action=edit&amp;section=4" title="Aldatu atal hau: «Paisaia industrialak»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h2><p>Industria-fabrikek euren bolumena dela eta, eta baita ere elkarrekin biltzeko duten joeragatik, eragin nabarmena dute <a href="/wiki/Paisaia" title="Paisaia">paisaian</a>. Gehienetan, badiak eta bailarak izan dira industria-fabriken kokagune nagusia, aurrez inguru hauetan zeuden azpiegiturak profitatuz. Lurralde hauetan industriak txertatzeko modu ezberdinak erakutsi ditu, paisaia industrial ezberdinak sortuz.</p><p>Historian zehar paisaia industrialak <b>herri beltzak</b> izenekoetatik <b>teknopolis</b> izenekoetara bilakatu dira, tartean <b>hirigune industrialak</b> osatuz. Lehenengo iraultza industrialean, <a href="/wiki/Harrikatz" title="Harrikatz">ikatza</a> zen energia-iturri nagusia eta horrela, fabrikak ikatz-<a href="/wiki/Meatzaritza" title="Meatzaritza">meategien</a> inguruan pilatzen ziren, kutsadura larria sortuz. Ondoan, langileen etxebizitzak egoten ziren, garraiorako auerak oso murritzak baitziren. <i><b>Black country</b></i> edo <b>herri beltz</b> deitu izan zaie horrelako lurraldeei, non langileak kondizio larrietan bizi ziren, esplotaturik eta kutsadura-giroan. <i>Black country</i> paisaiaren adibide paradigmatikoa <a href="/wiki/Birmingham" title="Birmingham">Birmingham</a> inguruko lurraldea izan zen <a href="/wiki/XIX._mendea" title="XIX. mendea">XIX. mendean</a>. <a href="/wiki/XX._mendea" title="XX. mendea">XX. mendean</a> zehar, <a href="/wiki/Kapitalismo" title="Kapitalismo">kapitalismoak</a> kontsumorako produktuak merkaturatu beharra izan zuenean, hiri populatuenen inguruan kokatu ziren, euren produktuei irtenbide zuzenena emateko, etorkinak erakarriz eta hiria handituz berriz ere. Gainera, ikatzaren ordez, petrolioa energia-iturri garrantzitsuena bilakatu zen eta horrela, ia ez zegoen beharrik fabrikak ikatz-meategi inguruetan kokatzeko. <b>Hirigune industrial</b> deritzenak sortu ziren horrela. Hirigune industrial hauen adibide gisa <a href="/wiki/Detroit" title="Detroit">Detroit</a>, <a href="/wiki/Madril" title="Madril">Madril</a> eta <a href="/wiki/Iru%C3%B1ea" title="Iruñea">Iruñea</a> <a href="/wiki/Euskal_Herria" title="Euskal Herria">Euskal Herrian</a> daude<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">[4]</a></sup>. <a href="/wiki/XX._mendea" title="XX. mendea">XX. mendearen</a> bukaeran, <a href="/w/index.php?title=Hirugarren_Industria_Iraultza&amp;action=edit&amp;redlink=1" class="new" title="Hirugarren Industria Iraultza (sortu gabe)">Hirugarren Industria Iraultzaren</a> haritik, fabrika mota berri bat garatzen da, teknologia aurreratuarekin eta prestakuntza maila handiko langileekin. Fabrika horiek gune jakin batzuetan biltzeko joera erakutsi dute, planifikazio zehatz bati jarraiki, botere politikoen babespean, elkarrekiko <a href="/w/index.php?title=Sinergia_(ekonomia)&amp;action=edit&amp;redlink=1" class="new" title="Sinergia (ekonomia) (sortu gabe)">sinergiak</a> baliatzeko asmoz eta hiriguneen inguruetan. <b>Teknopolis</b> edo <b>teknologia-parkeak</b> dira industria aurreratu hauek biltzen dituztenak<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">[5]</a></sup>.</p><p>Lurralde batean industria jarduera nagusia denean, fabrikak pilaturik azaltzen direlarik, paisaia <b>konplexu industrial</b> motakoa dela esaten da. Konplexu industrialak bailara batean edo portu baten inguruan kokatzen dira. Industria-fabrikak hain ugariak ez direnean, gehienetan hiri ertainen inguruan, <a href="/wiki/Industrialde" title="Industrialde">industrialdeetan</a> eta <a href="/w/index.php?title=Parke_teknologiko&amp;action=edit&amp;redlink=1" class="new" title="Parke teknologiko (sortu gabe)">parke teknologikoetan</a> biltzeko joera izaten dute. Azkenik, egungo garraio-sarearen garapenari esker, industria-fabrikak kokatzeko ez da hain faktore garrantzitsua garraio-kostua eta honenbestez fabrikak lurraldean sakabanatzeko joera hautematen da batzuetan.</p><div class="center"><div class="thumb tnone"><div class="thumbinner" style="width:402px;"><a href="/wiki/Fitxategi:Industria_paisaia01.svg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/6e/Industria_paisaia01.svg/400px-Industria_paisaia01.svg.png" width="400" height="150" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/6e/Industria_paisaia01.svg/600px-Industria_paisaia01.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/6e/Industria_paisaia01.svg/800px-Industria_paisaia01.svg.png 2x" data-file-width="665" data-file-height="250" /></a><div class="thumbcaption"><div class="magnify"><a href="/wiki/Fitxategi:Industria_paisaia01.svg" class="internal" title="Handitu"></a></div>Industriaren aglomerazioari buruzko industria-paisaia ezberdinak: <b>konplexu industrialak</b> industria asko (handiak eta ertainak batez ere) lurralde zehatz batean biltzen dituzten guneak dira; puntu zehatzetan bildutako fabrikak (ertainak eta txikiak, gehienetan) hirigune ezberdinetatik gertu kokatzen direnean, <b>industrialdeak</b> sortzen dira. Fabrikak lurralde osoan barreiaturik ere ager daitezke, <b>industria-paisaia sakabanatua</b> sortuz.</div></div></div></div><p><a href="/wiki/Euskal_Herria" title="Euskal Herria">Euskal Herrian</a>, <a href="/wiki/Bilboko_itsasadarra" title="Bilboko itsasadarra">Bilboko itsasadarra</a> izan da konplexu industrialaren adibide garbiena, <a href="/wiki/XX._mendea" title="XX. mendea">XX. mendearen</a> azken urteotan bertako industria asko desegin baziren ere. Bertako ertzeetan <a href="/wiki/Bizkaiko_Labe_Garaiak" title="Bizkaiko Labe Garaiak">Bizkaiko Labe Garaiak</a>, <a href="/wiki/La_Naval_ontziola" title="La Naval ontziola">La Naval ontziola</a> eta beste industria asko daude, askotan elkarren osagarriak. Hirigune industrialen adibide gisa, <a href="/wiki/Gasteiz" title="Gasteiz">Gasteiz</a>, <a href="/wiki/Gamarra_(argipena)" class="mw-disambig" title="Gamarra (argipena)">Gamarrako</a> industrialdearekin, eta <a href="/wiki/Iru%C3%B1ea" title="Iruñea">Iruñea</a>, <a href="/wiki/Landaben" title="Landaben">Landabengo</a> industrialdearekin, aipa daitezke. Gipuzkoan batez ere eta ibar estuetako ibilbideetan pilaturiko fabrikez osaturiko hirigune industrial berezien adibideak, herrietan bertan edo hauetatik oso gertu, <a href="/wiki/Eibar" title="Eibar">Eibar</a>, <a href="/wiki/Metalgintza" class="mw-redirect" title="Metalgintza">metalgintzako</a> fabrikekin, eta <a href="/wiki/Beasain" title="Beasain">Beasain</a> dira, azken hau <a href="/wiki/Construcciones_y_Auxiliar_de_Ferrocarriles" title="Construcciones y Auxiliar de Ferrocarriles">CAF</a> fabriken inguruan<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">[6]</a></sup>. <a href="/wiki/Bizkaiko_Zientzia_eta_Teknologia_Parkea" title="Bizkaiko Zientzia eta Teknologia Parkea">Bizkaiko Parke Teknologikoa</a>, <a href="/wiki/Zamudio" title="Zamudio">Zamudion</a> kokatua, Euskal Herriko teknopolis handiena da.</p><h2><span class="mw-headline" id="Industriaren_kontzentrazioa">Industriaren kontzentrazioa</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Industria&amp;veaction=edit&amp;section=5" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Industriaren kontzentrazioa»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Industria&amp;action=edit&amp;section=5" title="Aldatu atal hau: «Industriaren kontzentrazioa»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h2><p>Industriaren kontzentrazioa <a href="/wiki/Enpresa" title="Enpresa">enpresa</a> edo enpresa-talde batek hainbat industria bere mendean hartzen duteneko fenomenoa da. Egun, ugari dira industria ezberdinen elkartzeak eta erosketak, jarduera bera duten enpresen artean, elkar loturik dauden enpresen artean zein elkarrekin zer ikusirik ez duten jardueretan. <b>Kontzentrazio horizontala</b> izango da, enpresa bakar batek <i>industria-jarduera bereko</i> fabrikak edo enpresak biltzen dituenean. Kontzentrazio horizontala burutzearen arrazoiak merkatuan posizio menderatzaile bat osatzea (<a href="/wiki/Prezio" title="Prezio">prezioak</a> kontrolatuz neurri batean) edota <a href="/wiki/Eskala-errendimendu" title="Eskala-errendimendu">eskala-ekonomiak</a> sortzea, tamaina egokiago bat lortu eta kostuak murrizteko. <b>Kontzentrazio bertikala</b> <i>ekoizpen-prozesu berean</i> kokatzen diren industrien elkartzea da, lehengai eta produktu erdibukatuen hornikuntza modu egokian ziurtatu eta industria-taldeko tarteko fabriken merkatua bermatuz horrela. Elkartzen diren industriek zuzenean loturik ez daudenean, <b>konglomeratua</b> osatu dela esaten da<sup id="cite_ref-7" class="reference"><a href="#cite_note-7">[7]</a></sup>.</p><p>Aldi berean, <a href="/w/index.php?title=Cluster&amp;action=edit&amp;redlink=1" class="new" title="Cluster (sortu gabe)">cluster</a> edo industria-sare izeneko industria-enpresa multzoak ere osatzen dira. Cluster direlakoetan ezaugarri komunak (bezeroak, hornitzaileak, teknologia) dituzten industria biltzen dira enpresa-estrategiak, erakunde publikoekiko harremanak eta azpiegiturak partekatzeko, besteak beste. Cluster hauen helburua kostuak murriztu, konkurrentzia-ahalmena handitu eta <a href="/wiki/Sinergia" title="Sinergia">sinergiak</a> sortzea da <sup id="cite_ref-8" class="reference"><a href="#cite_note-8">[8]</a></sup>.</p><h2><span class="mw-headline" id="Industriaren_ekoizpen-metodoak">Industriaren ekoizpen-metodoak</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Industria&amp;veaction=edit&amp;section=6" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Industriaren ekoizpen-metodoak»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Industria&amp;action=edit&amp;section=6" title="Aldatu atal hau: «Industriaren ekoizpen-metodoak»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h2><div class="thumb tright"><div class="thumbinner" style="width:252px;"><a href="/wiki/Fitxategi:Krispy_Kreme_Doughnuts.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/5/50/Krispy_Kreme_Doughnuts.jpg/250px-Krispy_Kreme_Doughnuts.jpg" width="250" height="188" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/5/50/Krispy_Kreme_Doughnuts.jpg/375px-Krispy_Kreme_Doughnuts.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/5/50/Krispy_Kreme_Doughnuts.jpg/500px-Krispy_Kreme_Doughnuts.jpg 2x" data-file-width="2816" data-file-height="2112" /></a><div class="thumbcaption"><div class="magnify"><a href="/wiki/Fitxategi:Krispy_Kreme_Doughnuts.jpg" class="internal" title="Handitu"></a></div>Masako ekoizpenerako industriak ekoizpen-kateetan oinarritzen dute bere lana. Langile bakoitzak zeregin zehatza du <b>ekoizpen-katean</b> eta lan egiteko erritmoa askotan ez du berak kontrolatzen.</div></div></div><p>Industria Iraultza aurreko <a href="/wiki/Eskulangintza" title="Eskulangintza">artisauek</a> produktuaren ekoizpen-prozesu osoa, goitik behera, beraiek egiten zuten, eskuko lanaz nagusiki eta urtetan bildutako esperientziari esker. Industria-iraultzak industriara teknologiaren erabilera intentsiboa edo makinismoa ekartzeaz gainera fabriketako lan-antolakuntza metodo berriak ekarri zituen, espezializazioan eta kontrolean oinarriturik. Artisauak burutzen zuen lanean ez bezala, masako ekoizpenak <a href="/wiki/Produktibitate" title="Produktibitate">produktibitatearen</a> gehikuntza eskatzen zuen eta horretarako espezializazio eta <a href="/wiki/Lanaren_zatiketa" title="Lanaren zatiketa">lanaren zatiketarako</a> metodoak asmatu ziren, <a href="/wiki/Taylorismo" title="Taylorismo">taylorismo</a> izeneko antolaketa-modua osatuz. Horrela, langile bakoitzari zeregin zehatza emanez fabrikan, langileak bere lana burutzeko behar duen denboraren kontrol zorrotzarekin batera. <a href="/wiki/XX._mendea" title="XX. mendea">XX. mendearen</a> hasieran <a href="/wiki/Ekoizpen-lerro" class="mw-redirect" title="Ekoizpen-lerro">ekoizpen-lerroak</a> garatu ziren, non produktua ekoizteko fase guztiak bata bestearen ondoren kokatzen diren. <a href="/wiki/Ekoizpen-kate" class="mw-redirect" title="Ekoizpen-kate">Ekoizpen-kateak</a> <a href="/wiki/Henry_Ford" title="Henry Ford">Henry Ford</a> automobilgintza-enpresaburuak garatu zituen <a href="/wiki/XX._mendea" title="XX. mendea">XX. mendearen</a> hasieran. <a href="/wiki/Fordismo" title="Fordismo">Fordismoak</a> produktu homogeneo baten serie handiak ekoizteko beharra ikusten zuen gainera kostuak murriztu, ekoizpena gehitu eta etekin handiagoak lortzeko. <a href="/wiki/Taylorismo" title="Taylorismo">Taylorismoan</a>, fabrika barneko antolaketa zen helburu nagusia, denborak eta kostuak murriztuz; <a href="/wiki/Fordismo" title="Fordismo">fordismoan</a>, merkatua ere jartzen du bere ikusmiran, merke eskaini, asko saldu eta etekin oparoak lortzeko. Horrela, taylorismo ekoizpen-ekonomia bat den bitartean, fordismoa, eskaria-ekonomia bat da funtsean<sup id="cite_ref-9" class="reference"><a href="#cite_note-9">[9]</a></sup>.</p><p>XX. mendearen bukaeran, <a href="/wiki/Fordismo" title="Fordismo">fordismoa</a> baztertu eta <a href="/wiki/Toyotismo" title="Toyotismo">toyotismo</a> izeneko antolaketa-metodoa garatzen da industrian, aldi berean <a href="/wiki/Robotika" title="Robotika">robotika</a> eta <a href="/wiki/Automatizazio" title="Automatizazio">automatizazioa</a> garatu direlarik. <a href="/wiki/Japonia" title="Japonia">Japoniako</a> <a href="/wiki/Toyota" title="Toyota">Toyota</a> fabriketan abiarazitako <a href="/wiki/Toyotismo" title="Toyotismo">toyotismoa</a> <a href="/wiki/Fordismo" title="Fordismo">fordismoa</a> baino malguagoa da, fabrikan ekoizpen-kate bat izan daitekeen arren, langile-taldeek, <a href="/wiki/Kalitate-zirkulu" title="Kalitate-zirkulu">kalitate-zirkulu</a> moduan antolaturik askotan, langileek denboraren kontrol handiagoa izan eta tarea anitzagoak egiten baitituzte, edozein unetan langileak geldiarazi ditzakeen makineriari esker, prozesuari eta produktuari buruz duten ezagutza profitatuz <a href="/wiki/Kalitate" title="Kalitate">kalitatea</a> hobetzeko. <a href="/wiki/Just-in-time" title="Just-in-time">Just-in-time</a>, biltegietako tamaina gutxitzeko; <a href="/wiki/Kanban" title="Kanban">kanban</a>, ekoizpena era ordenatuan burutzeko; <a href="/wiki/Kaizen" title="Kaizen">kaizen</a>, kalitatearen hobekuntzarako eta <a href="/wiki/Poka-yoke" title="Poka-yoke">poka-yoke</a> segurtasunerako dira, besteak beste, ekoizpen-antolaketa berriaren tresna zenbait.</p><h2><span class="mw-headline" id="Industrialismoaren_ondorioak">Industrialismoaren ondorioak</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Industria&amp;veaction=edit&amp;section=7" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Industrialismoaren ondorioak»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Industria&amp;action=edit&amp;section=7" title="Aldatu atal hau: «Industrialismoaren ondorioak»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h2><p><a href="/wiki/XIX._mendea" title="XIX. mendea">XIX</a> eta <a href="/wiki/XX._mendea" title="XX. mendea">XX. mendeetan</a> izandako hazkunde ekonomikoa industriaren garapenari zor zaio neurri handienean eta egun ere herrialde askotako ekonomia-jarduera nagusia izaten jarraitzen du. Garapenerako jarduera eraginkorrena izatetik urrun, industriaren garapenak ideologia zehatz bati jarraitzen dio, <a href="/w/index.php?title=Industrialismo&amp;action=edit&amp;redlink=1" class="new" title="Industrialismo (sortu gabe)">industrialismo</a> izenekoa, <a href="/wiki/Kapitalismo" title="Kapitalismo">kapitalismoarekin</a> estu loturik:</p><table style="background-color:#efefef;border-collapse:collapse;border-radius: 6px;border-style:none;margin: .5em 2.75em; padding: 20px;"><tr><td width="20" valign="top" style="color:silver;font-size:40px;font-family:'Times New Roman',serif;font-weight:bold;text-align:left;padding-left:10px; padding-right:10px;">«</td><td align="left" style="padding:10px; padding-bottom:0; font-size: 90%;">Ekoizpen-sistema feudala, non ekoizpena artisau-elkarteen esku zegoen, ia ez zen nahikoa merkatu berriaren beharretarako. Gizarte arloan ez ezik, ingurumenean ere ondorio larriak ekarri zituen. data-file-width="958" data-file-height="704" /></a><div class="thumbcaption"><div class="magnify"><a href="/wiki/Fitxategi:Israel_DeadSea_Factories_chimneys(center)_-_Mar2009.JPG" class="internal" ekarri zuen ekoizpen-faktoreen erabileran. Horietarik, natura baliabideak izan ziren modu intentsiboan erabiltzen hasi ziren faktoreetako bat. Horren ondorioak nabarmenak izan dira: industriarako beharrezkoak diren lehengai naturalen gehiegizko erauzketak (<a href="/wiki/Basogintza" title="Basogintza">basogintza</a> intentsiboa, esaterako), ekoizpen-prozesuetan inguruko aire, ur eta lurraren kutsadura, kasu askotan inguruko gizakien bizitza-egoera nabarmen kaltetuz eta beste batzuetan osasun-larriak sortuz (erakusgarri nabarmenen artean, Indiako Bophal hirian industria kimiko bateko gas pozoitsu baten ihesi batek milaka lagun hil zituen 1984 urtean, <a href="/wiki/1986" title="1986">1986</a> urtean zentral nuklear batek eragindako <a href="/wiki/Txernobylgo_hondamendia" title="Txernobylgo hondamendia">Txernobylgo hondamendiak</a> ere giza eta ingurumen-ondorio larriak ekarri zituen) eta industria-produktuak kontsumitu ondoren naturan isuri edo botatzearen ondorioak (zabortegiak, esaterako).</p><p><a href="/wiki/1970eko_hamarkada" title="1970eko hamarkada">1970eko hamarkadatik</a> industriak dakartzan ingurumen-arazoek erakunde ofizialen arreta erakarri zuen. <a href="/wiki/Erromako_Kluba" title="Erromako Kluba">Erromako Kluba</a> izeneko aditu-taldeak hazkunde industrialak dakartzan kutsadura eta baliabide naturalen urritzea nabarmendu zituen. Geroztik eta askotan herriaren ekimenez <sup id="cite_ref-10" class="reference"><a href="#cite_note-10">[10]</a></sup>, gobernuek kezka horiek bereganatu eta ingurumenarekiko errespetua dakarten politika industrialak eratu dituzte. Nolanahi ere, askotan industria kutsatzaileen <a href="/wiki/Deslokalizazio" title="Deslokalizazio">lekualdatze</a> hutsa gertatu da, arrisku industriala zuten jarduerak eta hondakin toxikoak <a href="/wiki/Hirugarren_Mundua" title="Hirugarren Mundua">Hirugarren Munduko</a> herrialdeetara eramanez <sup id="cite_ref-11" class="reference"><a href="#cite_note-11">[11]</a></sup>. XXI mendearen hasieran, kezka hau mundu osora zabaldu da, industriak <a href="/wiki/Klima_aldaketa" title="Klima aldaketa">aldaketa klimatikoan</a> duen eragina dela eta. Horrela, <a href="/wiki/Garapen_iraunkor" title="Garapen iraunkor">garapen iraunkor</a> kontzeptua ere definitu da, industria-ekonomia ingurumenarekiko erlazio berri bat bilatu nahian besteak beste.</p><h3><span class="mw-headline" id="Industria_eta_gizartea">Industria eta gizartea</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Industria&amp;veaction=edit&amp;section=9" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Industria eta gizartea»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Industria&amp;action=edit&amp;section=9" title="Aldatu atal hau: «Industria eta gizartea»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h3><p><a href="/wiki/XIX._mendea" title="XIX. mendea">XIX. mendeko</a> hiri industrialetan langileak egoera larrian bizi ziren. Ordu askotako lanegunak burutu behar zituzten eta zigorrak jasaten zituzten nagusiak euren errendimendua nahikoa ez zela uste zuenean. Soldatak ere oso kaskarrak ziren. Auzo txiroetan bizi ziren, zerbitzu publiko eskasekin. Industriaren garapenak behar zuen inbertsio-tasa altuak errendimenduak ere altuak izatera behartzen zuen eta horren zama langileen gainean jartzen zen. XIX. mendeko langile industrialen bizimodua irudikatzen duen lan famatua <a href="/wiki/Charles_Dickens" title="Charles Dickens">Charles Dickensek</a> idatziriko <i>Hard times</i> (<i>Garai latzak</i>, euskaraz) eleberria da. Langileen egoera larri honek pentsamendu sozialista eta marxista garatu zuen, langileak industrialismoaren gehiegikerien aurka antolatzen ziren bitartean. Langilea beste makina bat balitz bezala hartzen zuen industriako metodo zientifikoak, <a href="/wiki/Alienazio" title="Alienazio">alienazio</a> kontzeptua sortzera eraman zuen teoria marxistaren baitan: antzinako artisuak ez bezala, langileek euren lanaren fruituarekin duten harremana arrotza da, langilea bere gizatasuna galduz horrela. Azken hamarkadetan aldatu egin da industrialismoak gizartean duen eragina. Langileen baldintzak, oro har, hobetu egin dira <a href="/wiki/Lehenengo_mundu" class="mw-redirect" title="Lehenengo mundu">lehenengo munduan</a>, teknologiaren produktibitate-aurrerapenei eta <a href="/wiki/Ongizate-estatu" title="Ongizate-estatu">ongizatearen estatuaren</a> zabalkuntzari esker. Industriaren eta ekonomia osoaren euskarri den <a href="/wiki/Kontsumismo" title="Kontsumismo">kontsumismoa</a> bilakatu da, ordea, gizakiaren alienazio iturri nagusi. Horren inguruan, azpimarratzekoak dira, beste batzuen artean, <a href="/wiki/Herbert_Marcuse" title="Herbert Marcuse">Herbert Marcuse</a> filosofo eta soziologoaren lanak <sup id="cite_ref-12" class="reference"><a href="#cite_note-12">[12]</a></sup>.</p><h2><span class="mw-headline" id="Industria_arloz_arlo">Industria arloz arlo</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Industria&amp;veaction=edit&amp;section=10" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Industria arloz arlo»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Industria&amp;action=edit&amp;section=10" title="Aldatu atal hau: «Industria arloz arlo»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h2><div class="thumb tright"><div class="thumbinner" style="width:402px;"><a href="/wiki/Fitxategi:Egurindustria01.svg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/a0/Egurindustria01.svg/400px-Egurindustria01.svg.png" width="400" height="366" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/a0/Egurindustria01.svg/600px-Egurindustria01.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/a0/Egurindustria01.svg/800px-Egurindustria01.svg.png 2x" data-file-width="2516" data-file-height="2303" /></a><div class="thumbcaption"><div class="magnify"><a href="/wiki/Fitxategi:Egurindustria01.svg" class="internal" title="Handitu"></a></div>Egur-industriako azpiarloen arteko loturak (basogintza, ordea, ez da industria arloan kokatzen). Industria-arloek <b>balio-kate bat</b> osatzen duten elkarren artean. Horrela, produktu bukatuan denek dute parte bat.</div></div></div><p>Industrian arlo ezberdinak bereizten badira ere (automobilgintza, zementua, elektronika-produktuak, ...), industria-produkzioaren balioa kate moduan osatzen da industria ezberdinen artean, lehengaietatik hasi, oinarrizko gaiak, produktu erdibukatuak eta osagaiak tartean eta kontsumo-produktuekin bukatzeko.</p><p>Industria-jardueren oinarrian <a href="/wiki/Meatzaritza" title="Meatzaritza">meagintza</a> eta <a href="/w/index.php?title=Energia-industria&amp;action=edit&amp;redlink=1" class="new" title="Energia-industria (sortu gabe)">energia-industria</a> daude, nekazaritza-sektoreak ematen duen hainbat lehengairekin batera. <a href="/wiki/Meatzaritza" title="Meatzaritza">Meagintza</a> makineria astuna behar duen sektorea izaten da. Aldi berean, kokapen zehatzak dituen jarduera da, meategiak non, han kokatuko baitira meakgintza-fabrikak, <a href="/wiki/Mea" title="Mea">mea</a> erauzi eta haren oinarrizko eraldaketa egiteko.</p><p><a href="/wiki/Energia" title="Energia">Energiaren</a> sorrera eta banaketa burutzen dituen <a href="/w/index.php?title=Energia-industria&amp;action=edit&amp;redlink=1" class="new" title="Energia-industria (sortu gabe)">energia-industria</a> da oinarrizko industriarik garrantzitsuena, beste ekonomia-arloek, industria guztiak barne, energia hornikuntzaren mendean baitaude. Energia-industriek azpiegitura eta instalakuntza konplexuak behar dituzte eta enpresa handien esku egoten da. Energia-iturria zein den, jarduerak ezaugarri ezberdinak ditu, ordea. Adibidez, petrolioaz sortutako energia-fabrikak petrolioaren hornikuntza bermatzen duten azpiegituren inguruetan daude (portuak, oliobideak). Zentral nuklearrek, berriz, ur-hornikuntza ziurra duten tokietan kokatu behar dira; erregai gisa erabiltzen duten <a href="/wiki/Uranio" title="Uranio">uranioaren</a> zuzeneko hornikuntza, ordea, ez da hain erabakigarria hauen kasuan. Garapen bidean dauden energia berriztagarrien industriak, berriz, iturburu-mota zein den kokatuko dira (haize handia jotzen duen tokietan kokatuko dira energia eolikoaren instalakuntzak, ibaietan energia hidraulikoa, toki eguzkitsuetan eguzki-energia). Energia berriztagarrien instalakuntzetarako egin beharreko gutxieneko inbertsioa petrolioaren eta energiaren nuklearraren kasuan baino askoz txikiagoa da, euren errendimendua txikiagoa dela kontuan hartu behar bada ere.</p><p>Ekoizpen-kate industrialeko bigarren fase batean, eraldaketa-industriak kokatzen dira: <a href="/wiki/Metalgintza" class="mw-redirect" title="Metalgintza">metalgintza</a> eta lehengaietatik oinarrizko eta bitarteko gaiak ekoizten dituzten bestelako industriak daude (<a href="/wiki/Kautxu" title="Kautxu">kautxua</a>, <a href="/wiki/Plastiko" title="Plastiko">plastikoak</a>, <a href="/wiki/Zementu" title="Zementu">zementua</a> eta bestelako gaiak). Industria hauek ekoiztutako gaiak beste industrietarako izaten dira eta erabilera anitza dute. Hori dela eta, bere ekoizpena masiboa izaten da eta fabrika handietan burutzen da, oso gai berezien kasuan ezik (egurraren eraldaketan, esaterako). Eraldaketa-maila ezberdinak daude gainera: adibidez, <a href="/wiki/Metalgintza" class="mw-redirect" title="Metalgintza">metalgintza</a> sektorean burdina gordina ekoizten duen <a href="/wiki/Burdingintza" title="Burdingintza">siderurgia</a>, <a href="/wiki/Labe_garai" title="Labe garai">labe garaiak</a> (mea burdina bihurtzeko) eta <a href="/w/index.php?title=Laminazio-tren&amp;action=edit&amp;redlink=1" class="new" title="Laminazio-tren (sortu gabe)">laminazio-trenak</a> (burdina gordinari txapa edo bestelako eitea emateko) izanik bere instalakuntza nagusiak, eta metalaren eraldaketa xeheagoa (material mota jakinak eta zehatzak eraldatzen dituzten tailer txikiak) bereizten dira. Hala ber, <a href="/w/index.php?title=Kimika-industria&amp;action=edit&amp;redlink=1" class="new" title="Kimika-industria (sortu gabe)">kimika-industrian</a> <a href="/w/index.php?title=Birfindegi&amp;action=edit&amp;redlink=1" class="new" title="Birfindegi (sortu gabe)">birfindegiak</a> dira fabrika nagusiak eta bigarren maila batean petrolioa jatorri ez duten gai kimikoak edo petroliotik birfindutako oinarrizko gaiak ekoizten dituzten beste kimika-fabrika batzuk ere badaude atal honetan (<a href="/wiki/Plastiko" title="Plastiko">plastikoak</a>, <a href="/w/index.php?title=Zuntz_sintetiko&amp;action=edit&amp;redlink=1" class="new" title="Zuntz sintetiko (sortu gabe)">zuntz sintetikoak</a> eta <a href="/wiki/Ongarri" title="Ongarri">ongarri</a> kimikoak ekoizten dituztenak, esaterako). Beste alde batetik, siderurgiak eta kimika-industriak energia asko behar izaten dute euren jardueretako; hau dela eta, arlo hauetako fabrika handi askotan ekoizpen-prozesua etenik gabe garatzen da gau eta egun, prozesua hutsetik abiarazteak ekarriko lukeen energia-bultzada aurrezteko. Eraldaketa-maila honetan gero eta garrantzi handiagoa duten birziklapen-industriak ere barneratu behar dira, <a href="/w/index.php?title=Papergintza&amp;action=edit&amp;redlink=1" class="new" title="Papergintza (sortu gabe)">papergintza</a>, <a href="/w/index.php?title=Beiragintza&amp;action=edit&amp;redlink=1" class="new" title="Beiragintza (sortu gabe)">beiragintza</a> eta <a href="/w/index.php?title=Plastikogintza&amp;action=edit&amp;redlink=1" class="new" title="Plastikogintza (sortu gabe)">plastikogintzan</a> batez ere. Siderurgian ere ohikoa da meaz gainera <a href="/wiki/Txatar" title="Txatar">txatarra</a> erabiltzea metal berria ekoizteko.</p><p>Hirugarren fase batean koka daitezke <a href="/wiki/Ekipo-ondasun" title="Ekipo-ondasun">ekipo-ondasunen</a> industriak, kapital-industria izenez ere ezagunak. <a href="/wiki/Automobilgintza" title="Automobilgintza">Automobilgintza</a>, <a href="/wiki/Makina-tresna" title="Makina-tresna">makina-tresna</a>, <a href="/w/index.php?title=Aeronautika-industria&amp;action=edit&amp;redlink=1" class="new" title="Aeronautika-industria (sortu gabe)">aeronautika-industria</a>, <a href="/w/index.php?title=Ontzigintza&amp;action=edit&amp;redlink=1" class="new" title="Ontzigintza (sortu gabe)">ontzigintza</a> dira sail honetan biltzen diren sektore nagusietako batzuk. Industria hauetan osagai zehatzen ekoizpena eta ondorengo muntaia<sup id="cite_ref-13" class="reference"><a href="#cite_note-13">[13]</a></sup> dira fabrikako jarduera nagusiak, horretan trebetasun berezia duen lan-indarra erabiliz. Sektore hauetako produktuak epe luzerako <a href="/wiki/Inbertsio" title="Inbertsio">inbertsio</a> moduan erabiltzen dituzte beste industriek. Hori dela eta, industria osoko bilakaeraren aurreikuspen gisa erabiltzen da industria hauetako egoera.</p><table class="navbox" cellspacing="0" style=";"><tr><td style="padding:2px;"><table cellspacing="0" class="nowraplinks collapsible autocollapse" style="width:100%;background:transparent;color:inherit;;"><tr><th style=";" colspan="2" class="navbox-title"><div style="float:left; width:6em;text-align:left;"></div><span class="" style="font-size:110%;">Industria-arloak</span></th></tr><tr style="height:2px;"><td></td></tr><tr><td class="navbox-group" style=";;">Oinarrizko industriak</td><td style="text-align:left;border-left-width:2px;border-left-style:solid;width:100%;padding:0px;;;" class="navbox-list navbox-odd"><div style="padding:0em 0.25em"><a href="/w/index.php?title=Energia-industria&amp;action=edit&amp;redlink=1" class="new" title="Energia-industria (sortu gabe)">Energia-industria</a> - Iraultza Industriala <a href="/wiki/Ingalaterra" title="Ingalaterra">Ingalaterratik</a> abian jarri eta handik <a href="/wiki/Mendebaldeko_Europa" title="Mendebaldeko Europa">Europako mendebaldean</a> eta <a href="/wiki/Ipar_Amerika" title="Ipar Amerika">Ipar Amerikan</a> soilik zabaldu zen hasiera batean. <a href="/wiki/XX._mendea" title="XX. mendea">XX. mendean</a> zehar, <a href="/wiki/Sobietar_Errepublika_Sozialisten_Batasuna" title="Sobietar Errepublika Sozialisten Batasuna">Sovietar Batasuna</a>, <a href="/wiki/Japonia" title="Japonia">Japonia</a> eta Europako beste zenbait herrialde ere (<a href="/wiki/Espainia" title="Espainia">Espainia</a> eta <a href="/wiki/Italia" title="Italia">Italia</a>, esaterako) industria-jarduera garrantzitsu bat garatzen hasi ziren. Horrela, herrialde industrializatu eta ez-industrializatuen sailkapena egin zen, herrialde bakoitzean industriak zuen pisuari buruz, tartean industrializazio bidean dauden herrialdeak ere aipatuz. Horrela, nazioarteko ekonomia-egitura nahiko garbia izan zen XX. mendean, herrialde industrializatuak <a href="/wiki/Ipar_hemisferio" title="Ipar hemisferio">ipar hemisferioan</a> eta herrialde ez-industrializatuak <a href="/wiki/Hego_hemisferio" title="Hego hemisferio">hego hemisferioan</a> kokatuz. XX. mendearen bukaera aldera, <a href="/wiki/Ekialdeko_Asia" title="Ekialdeko Asia">Ekialdeko Asiako</a> herrialdeen industrializazio-prozesua azkartu (<a href="/wiki/Hego_Korea" title="Hego Korea">Hego Korea</a>, <a href="/wiki/Taiwan" title="Taiwan">Taiwan</a>, <a href="/wiki/Singapur" title="Singapur">Singapur</a> eta, batez ere, <a href="/wiki/Txinako_Herri_Errepublika" title="Txinako Herri Errepublika">Txinan</a>) eta munduan industria-herrialde garrantzitsu bilakatu dira. Aldi berean, herrialde industrializatu tradizionalen lerratze nabarmen bat izan zen zerbitzuen ekonomiaruntz, herrialde industrializatu berrien produktu merkeen konkurrentziari aurre egin ezinik eta etekin handiagoak ematen dituzten sektoreak garatuz (<a href="/wiki/Finantzak" title="Finantzak">finantzak</a>, <a href="/wiki/Bioteknologia" title="Bioteknologia">bioteknologia</a>) . Ildo honetatik, <a href="/wiki/Txinako_Herri_Errepublika" title="Txinako Herri Errepublika">Txina</a> <i>munduko fabrika</i> bilakatu dela aipatu da <sup id="cite_ref-14" class="reference"><a href="#cite_note-14">[14]</a></sup>. Hego hemisferioko herrialde batzuk ere, <i>herrialde gainerakor</i> izenpean (<a href="/wiki/India" title="India">India</a>, <a href="/wiki/Brasil" title="Brasil">Brasil</a>), industria-garapen nabarmen bati ekin diote XXI. mendearen hasieran, petrolioaren eta beste lehengai eta nekazal produktuen ustiapenaren inguruan kasu batzuetan eta siderurgia bezalako industria-jarduera helduekin loturik beste batzuetan, industrializazio hau herrialde garatuen mendekotasunezko harreman batez garatu izan bada ere, <a href="/wiki/Multinazional" title="Multinazional">multinazionalen</a> bitartez esaterako.</p><p>Hala ere, egun Europako mendebalde osoan industria-jarduerak garrantzi handia izaten jarraitzen du. Bertako industria-eskualde nagusienetakoa da <a href="/wiki/Alemania" title="Alemania">Alemanian</a> kokatzen da, <a href="/wiki/Ruhr" title="Ruhr">Ruhr</a> ibaiaren arroaren inguruan batez ere. <a href="/wiki/Saxonia" title="Saxonia">Saxonian</a>, Frantziako iparraldean eta <a href="/wiki/Rh%C3%B4ne" title="Rhône">Rodano</a> ibaiaren arroan eta <a href="/wiki/Po" title="Po">Po</a> ibaiaren inguruan <a href="/wiki/Italia" title="Italia">Italian</a> kokatzen dira beste industria-eskualde garrantzitsu batzuk. <a href="/wiki/Ipar_Amerika" title="Ipar Amerika">Ipar Amerikan</a>, <a href="/wiki/Ameriketako_Estatu_Batuak" title="Ameriketako Estatu Batuak">Estatu Batuen</a> ipar-ekialdean da industria-eskualde nagusia, <a href="/wiki/Aintzira_Handiak" title="Aintzira Handiak">Aintzira Handien</a> inguruan. <a href="/wiki/Kalifornia" title="Kalifornia">Kalifornian</a> industria aurreratuak dituen eskualde bat da. <a href="/wiki/Asiako_Ekialdea" class="mw-redirect" title="Asiako Ekialdea">Asiako Ekialdean</a>, <a href="/wiki/Japonia" title="Japonia">Japonia</a>, <a href="/wiki/Hego_Korea" title="Hego Korea">Hego Korea</a>, <a href="/wiki/Taiwan" title="Taiwan">Taiwan</a> eta <a href="/wiki/Hong_Kong" title="Hong Kong">Hong Kong</a> hiritik <a href="/wiki/Pekin" title="Pekin">Beijingera</a> doan eskualdea, <a href="/wiki/Txina" class="mw-redirect" title="Txina">Txinan</a> azken hau, dira industria-eskualde trinkoenak. <a href="/wiki/Hego_Amerika" title="Hego Amerika">Hego Amerikan</a> <a href="/wiki/S%C3%A3o_Paulo" title="São Paulo">São Paulo</a> hiri inguruko eskualdea industria anitz biltzen ditu.</p><p>2007 urteko datuen arabera, hauek ziren industria-ekoizpena handiena zuten 12 herrialdeak, modu ordenatuan: <a href="/wiki/Ameriketako_Estatu_Batuak" title="Ameriketako Estatu Batuak">AEB</a>, <a href="/wiki/Txinako_Herri_Errepublika" title="Txinako Herri Errepublika">Txina</a>, <a href="/wiki/Japonia" title="Japonia">Japonia</a>, <a href="/wiki/Alemania" title="Alemania">Alemania</a>, <a href="/wiki/Italia" title="Italia">Italia</a>, <a href="/wiki/Britainia_Handia" title="Britainia Handia">Britania Handia</a>, <a href="/wiki/Frantzia" title="Frantzia">Frantzia</a>, <a href="/wiki/Errusia" title="Errusia">Errusia</a>, <a href="/wiki/Brasil" title="Brasil">Brasil</a>, <a href="/wiki/Hego_Korea" title="Hego Korea">Hego Korea</a>, <a href="/wiki/Kanada" title="Kanada">Kanada</a>, <a href="/wiki/Espainia" title="Espainia">Espainia</a>, <a href="/wiki/Mexiko" title="Mexiko">Mexiko</a> eta <a href="/wiki/India" title="India">India</a>. Erlatiboki, <a href="/wiki/Ameriketako_Estatu_Batuak" title="Ameriketako Estatu Batuak">Estatu Batuek</a> munduko ekoizpen osoaren&#160;%28 hartzen zuten <a href="/wiki/1990" title="1990">1990ean</a>. <a href="/wiki/2008" title="2008">2008an</a>&#160;%24ra jeisi zen bere partaidetza. Txinak, berriz,&#160;%4tik&#160;%18ra igo du bere partaidetza 1990-2008 urteetan, <a href="/wiki/Japonia" title="Japonia">Japoniaren</a> industriak ekoizpen osoaren&#160;%22tik&#160;%14ra jeitsi den bitartean. Urte bitarte horretan, <a href="/wiki/Brasil" title="Brasil">Brasil</a> eta <a href="/wiki/India" title="India">India</a> herrialdeek ere nabarmen igo dute euren partaidetza<sup id="cite_ref-15" class="reference"><a href="#cite_note-15">[15]</a></sup><sup id="cite_ref-16" class="reference"><a href="#cite_note-16">[16]</a></sup>.</p><h2><span class="mw-headline" id="Euskal_Herrian">Euskal Herrian</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Industria&amp;veaction=edit&amp;section=12" class="mw-editsection-visualeditor" title="Aldatu atal hau: «Euskal Herrian»">aldatu</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Industria&amp;action=edit&amp;section=12" title="Aldatu atal hau: «Euskal Herrian»">aldatu iturburu kodea</a><span class="mw-editsection-bracket">]</span></span></h2><div style="font-style: italic; padding-left: 2em; margin-bottom: 0.5em;">Sakontzeko, irakurri: «<a href="/wiki/Industrializazioa_Euskal_Herrian" title="Industrializazioa Euskal Herrian">Industrializazioa Euskal Herrian</a>»</div><p><a href="/wiki/XX._mendea" title="XX. mendea">XX. mendearen</a> hasieran kokatzen da industriaren garapena <a href="/wiki/Euskal_Herria" title="Euskal Herria">Euskal Herrian</a>, <a href="/wiki/Bizkaia" title="Bizkaia">Bizkaian</a> eta <a href="/wiki/Gipuzkoa" title="Gipuzkoa">Gipuzkoan</a> batez ere. Ordura arte, Bizkaian burdina-meagintza jarduerako gaiak esportatzeari ekiten bazitzaion ere, laster hasiko ziren siderurgia-fabrikak garatzen. <a href="/wiki/1902" title="1902">1902</a> urtean <a href="/wiki/Bizkaiko_Labe_Garaiak" title="Bizkaiko Labe Garaiak">Bizkaiko Labe Garaiak</a> industria-enpresa handia sortu zen, inguruan siderurgia-enpresa txikiagoekin batera. Ohartu behar da burdina-ekoizpenaren zatirik handiena <a href="/wiki/Esportazio" title="Esportazio">esportatu</a> egiten zela Europako beste herrialdeetara. <a href="/wiki/XX._mendea" title="XX. mendea">XX. mendean</a> zehar ere nagusitu den sektorea izan da siderurgia, <a href="/wiki/Bilboko_itsasadarra" title="Bilboko itsasadarra">Bilboko itsasadarraren</a> inguruetan. Ontzigintzan <a href="/wiki/Euskalduna_ontziola" title="Euskalduna ontziola">Euskalduna</a> sortu zen <a href="/wiki/1900" title="1900">1900</a> urtean <a href="/wiki/Bilbo" title="Bilbo">Bilbon</a> bertan. Industria garapenaren lehenengo urte haietan, Bizkaian siderurgia nagusi zen bitartean, Gipuzkoan industria anitzagoa garatu zen, <a href="/w/index.php?title=Papergintza&amp;action=edit&amp;redlink=1" class="new" title="Papergintza (sortu gabe)">papergintzan</a> (<a href="/w/index.php?title=Papelera_del_Araxes&amp;action=edit&amp;redlink=1" class="new" title="Papelera del Araxes (sortu gabe)">Papelera del Araxes</a>, <a href="/wiki/1858" title="1858">1858</a>, Tolosan; <a href="/wiki/Papelera_Espa%C3%B1ola" class="mw-redirect" title="Papelera Española">Papelera Española</a>, <a href="/wiki/1901" title="1901">1901</a>, <a href="/wiki/Tolosa" title="Tolosa">Tolosan</a> eta <a href="/wiki/Errenteria" title="Errenteria">Errenterian</a>, esaterako), <a href="/w/index.php?title=Metalgintzan&amp;action=edit&amp;redlink=1" class="new" title="Metalgintzan (sortu gabe)">metalgintzan</a> (Construcciones Metálicas, 1901, <a href="/wiki/Beasain" title="Beasain">Beasainen</a>; <a href="/wiki/Uni%C3%B3n_Cerrajera" title="Unión Cerrajera">Unión Cerrajera</a>, <a href="/wiki/1902" title="1902">1902</a>, <a href="/wiki/Arrasate" title="Arrasate">Arrasaten</a> eta <a href="/wiki/1926" title="1926">1926an</a> <a href="/wiki/Bergara" title="Bergara">Bergaran</a>, esaterako), <a href="/wiki/Ehungintza" title="Ehungintza">ehungintzan</a> (<a href="/w/index.php?title=Textil_Movilla,_S.A.&amp;action=edit&amp;redlink=1" class="new" title="Textil Movilla, S.A. (sortu gabe)">Textil Movilla, S.A.</a>, <a href="/wiki/1902" title="1902">1902</a>, Bergaran, esaterako), <a href="/w/index.php?title=Larrugintza&amp;action=edit&amp;redlink=1" class="new" title="Larrugintza (sortu gabe)">larrugintza</a> Antzuolan, elikagai-industria (<a href="/w/index.php?title=La_Ib%C3%A9rica&amp;action=edit&amp;redlink=1" class="new" title="La Ibérica (sortu gabe)">La Ibérica</a>, <a href="/wiki/1866" title="1866">1866</a>, <a href="/wiki/Errenteria" title="Errenteria">Errenterian</a>, Olibet <a href="/wiki/Gaileta" title="Gaileta">gailetak</a> ekoizten zituena). Gipuzkoan ere aspalditik zegoen garatuta <a href="/wiki/Meatzaritza" title="Meatzaritza">meatzaritza</a>, <a href="/wiki/Zerain" title="Zerain">Zerainen</a>, <a href="/wiki/Legorreta" title="Legorreta">Legorretan</a>, <a href="/wiki/Irun" title="Irun">Irunen</a> eta <a href="/wiki/Oiartzun" title="Oiartzun">Oiartzunen</a> nagusiki. <a href="/wiki/Araba" title="Araba">Araba</a> industria iraultzatik kanpo geratuko zen hasiera batean, Bizkaia ondoko <a href="/wiki/Laudio" title="Laudio">Laudion</a> ezik. <a href="/wiki/Nafarroa" class="mw-disambig" inguruan. <a href="/wiki/Iparralde" class="mw-redirect mw-disambig" title="Iparralde">Iparraldean</a>, <a href="/wiki/1882" title="1882">1882</a>. urtean <a href="/wiki/Bokale_(Lapurdi)" title="Bokale (Lapurdi)">Bokalen</a> sortutako <a href="/wiki/Forges_de_l%27Adour" title="Forges de l'Adour">Forges de l'Adour</a> siderurgia-fabrika handia izan zen industria-gune bakarra, <a href="/wiki/Landak" Zehatzago, 1951-52 urtera arte ez zen gerra aurreko ekoizpen industrialaren maila berriz ere lortu<sup id="cite_ref-17" class="reference"><a href="#cite_note-17">[17]</a></sup>. Autarkia baztertu eta <a href="/wiki/1960ko_hamarkada" title="1960ko hamarkada">1960eko hamarkadaren</a> hasieratik hazkunde ekonomiko azkar bati ekin zitzaion, industriari esker batez ere. Hego Euskal Herria izan zen gainera <a href="/wiki/Espainia" title="Espainia">Espainia</a> osoan gehien garatu zen erregioetako bat, <a href="/wiki/Katalunia" title="Katalunia">Kataluniarekin</a> batera. Industria-lanak erakarrita, Espainiako nekazari-eskualdeetatik milaka eta milaka etorkin heldu ziren Hego Euskal Herrian osatzen ari ziren herri industrialetara <sup id="cite_ref-18" class="reference"><a href="#cite_note-18">[18]</a></sup>. <a href="/wiki/1973" title="1973">1973</a> urtean bukatu zen industriaren hazkunde-aldi hau. Baina ordurako Gipuzkoan, Bizkaian eta Araban (eta ordena horretan) industria zen ekonomiaren sektore nagusia, produkzio osoaren&#160;%50etik gora hartzen zuela. Nafarroan ere industria zen nagusi produkzio osoaren&#160;%40ko pisu batez. Espainia osoan, berriz, industria ez zen ekonomia osoaren&#160;%40ra heltzen<sup id="cite_ref-19" class="reference"><a href="#cite_note-19">[19]</a></sup><sup id="cite_ref-20" class="reference"><a href="#cite_note-20">[20]</a></sup>. <a href="/w/index.php?title=1973ko_petrolio-krisialdia&amp;action=edit&amp;redlink=1" class="new" title="1973ko petrolio-krisialdia (sortu gabe)">1973ko petrolio-krisialdiaren</a> ondorioz, euskal industriak kinka larrian egon zen <a href="/wiki/1980ko_hamarkada" title="1980ko hamarkada">1980ko hamarkadako</a> erdialdera arte. Petrolioaren krisialdiak kostuen igoera handia ekarri zuen eta fabrika askok itxi behar izan zuten, bere produktuei irteera eman eta inflazioak ekarritako finantza-zorren gehikuntza kitatu ezinik. Aldi berean, industriak egitura-arazo larriak zituen: beste herrialde batzuetan merkeago ekoizten ziren produktu zaharkituak ekoizten zituen (metalgintza sektorean, batik bat), azpiegitura-arazoak, kudeaketa desegokia eta gehiegizko langile-kopurua . Irtenbidea 1980ko hamarkadako <a href="/wiki/Industria-birmoldaketa" title="Industria-birmoldaketa">industria-birmoldaketa</a> izan zen. Gizarte-sektoreetatik kritikatua izan zen politika horren ondorioz, industria fabrika asko itxi eta milaka langile euren lanpostuetatik erauziak izan ziren <sup id="cite_ref-21" class="reference"><a href="#cite_note-21">[21]</a></sup> baina aldi berean ez zen sortu industria-sare berri eta sendo bat, gobernuetatik eskainitako laguntzak gorabehera<sup id="cite_ref-22" class="reference"><a href="#cite_note-22">[22]</a></sup>. 1985 urtetik industria berpizten hasi zen. Urte horietan <a href="/wiki/Espainia" title="Espainia">Espainia</a> <a href="/wiki/Europar_Batasuna" title="Europar Batasuna">Europar Batasunean</a> sartu izanak industria-produktuetarako merkatu berriak ireki eta industriarako sustapen tinkoagoa ekarri zuen. Industrian eragin nabarmena izan zuen 1992-93 urteetako nazioarteko beherakada ekonomikoaren ondoren, industriak indarberritu eta nabarmen egin zuen aurrera, nazioarteko suspertze ekonomiko nabarmenari eta eraikuntza sektoreak eragindako bultzadari esker<sup id="cite_ref-23" class="reference"><a href="#cite_note-23">[23]</a></sup>, gobernuetatik abiarazitako berrikuntzarako eta industria-sareak osatzeko <a href="/w/index.php?title=Klusterizazio&amp;action=edit&amp;redlink=1" class="new" title="Klusterizazio (sortu gabe)">klusterizaziorako</a> politika estrategikoak tarteko <sup id="cite_ref-24" class="reference"><a href="#cite_note-24">[24]</a></sup>, azpiegituren garapen nabarmen batekin batera. Ildo horretatik, <a href="/w/index.php?title=Altzairugintza&amp;action=edit&amp;redlink=1" class="new" title="Altzairugintza (sortu gabe)">altzairugintza</a> berezia, <a href="/wiki/Elektrogailu" class="mw-redirect" title="Elektrogailu">elektrogailuak</a>, <a href="/wiki/Ardogintza" title="Ardogintza">ardogintza</a>, <a href="/w/index.php?title=Papergintza&amp;action=edit&amp;redlink=1" class="new" title="Papergintza (sortu gabe)">papergintza</a>, <a href="/wiki/Makina_tresna" class="mw-redirect" title="Makina tresna">makina tresna</a>, automobilgintzarako osagaiak eta <a href="/wiki/Energia_eoliko" title="Energia eoliko">energia eolikoa</a> izan dira Hego Euskal Herrian bultzatu diren cluster industrial nagusiak.<sup id="cite_ref-25" class="reference"><a href="#cite_note-25">[25]</a></sup></p><p>Cluster bidezko espezializazioa eta <a href="/wiki/Globalizazio" title="Globalizazio">globalizazio</a>-inguru batean nazioarteratzea (esportazio-tasa nabarmenekin) dira egun euskal industriaren ezaugarri nagusiak, 2008ko krisialdi ekonomikoak aurreko urteotako baikortasuna zalantzan jarri badu ere. Egun, Hego Euskal Herriko industriak Europan baino pisu handiagoa du ekonomia osoan. <a href="/wiki/Iberdrola" title="Iberdrola">Iberdrola</a> energia sektorean, besteak beste <a href="/wiki/Pasaiako_portua" title="Pasaiako portua">Pasaiako portuko</a> zentral termikoarekin; <a href="/wiki/Petronor" title="Petronor">Petronor</a> petrokimikan, besteak beste <a href="/wiki/Muskiz" title="Muskiz">Muskizeko</a> birfindegiarekin eta <a href="/wiki/Mondragon_(korporazioa)" title="Mondragon (korporazioa)">MCC</a> kooperatiba industrialen multzoa dira talde industrial handienak. Bilboko itsasadarrean, altzairugintza arloan, eta <a href="/wiki/Iru%C3%B1ea" title="Iruñea">Iruñea</a> ondoko <a href="/wiki/Landabengo_industrialdea" class="mw-redirect" title="Landabengo industrialdea">Landabengo industrialdea</a>, automobilgintzaren inguruan. Aldi berean, fabrika txiki eta ertain asko agertzen dira barreiaturik lurraldean zehar, askotan industrialdeetan bilduta. Industrialdeetan, Gasteiz ondoko <a href="/w/index.php?title=Jundizeko_industrialdea&amp;action=edit&amp;redlink=1" class="new" title="Jundizeko industrialdea (sortu gabe)">Jundizeko industrialdea</a> eta Irungo <a href="/w/index.php?title=Zaisa_industrialdea&amp;action=edit&amp;redlink=1" class="new" title="Zaisa industrialdea (sortu gabe)">Zaisa industrialdea</a>, azpiegitura zabaletarako lotura zuzena dituztenak, agertzen dira adibide gisa.</p><h2><span class="mw-headline" id="Industria_jarduera_ekonomikoen_sailkapenetan">Industria jarduera ekonomikoen sailkapenetan</span><span
