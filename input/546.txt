latissimo, est summa omnium <a href="/wiki/Ars" title="Ars">artium</a> et <a href="/wiki/Scientia_(ratio)" title="Scientia (ratio)">scientiarum</a> et <a href="/wiki/Technologia" title="Technologia">technologiarum</a> quae de <a href="/wiki/Terra_(humus)" class="mw-redirect" title="Terra (humus)">terris</a> colendis et <a href="/wiki/Animal" class="mw-redirect" title="Animal">animalibus</a> creandis curant, ut <a href="/wiki/Pomum" class="mw-redirect" title="Pomum">poma</a>, <a href="/wiki/Frumentum" title="Frumentum">frumenta</a>, <a href="/w/index.php?title=Chara&amp;action=edit&amp;redlink=1" class="new" title="Chara (non est haec pagina)">charas</a>, <a href="/wiki/Caro" title="Caro">carnes</a>, <a href="/wiki/Textile" href="/w/index.php?title=Agricultio&amp;action=edit&amp;redlink=1" class="new" title="Agricultio (non est haec pagina)">agricultio</a><sup><small><a href="/wiki/Disputatio:Agricultura" title="Disputatio:Agricultura">?</a></small></sup> autem <a href="/wiki/Animal" class="mw-redirect" title="Animal">animalibus</a> creandis. </p><p>Hoc commentarium est de <a href="/wiki/Vita" title="Vita">vita</a> et communibus artibus agricolis, de quibus <a href="/wiki/Roma_antiqua" class="mw-redirect" title="Roma antiqua">Romani</a> antiqui magnopere curabant. In <a href="/wiki/Liber" title="Liber">libro</a> <a href="/wiki/De_agri_cultura" title="De agri cultura"><i>De Agri Cultura</i> (seu <i>De Re Rustica</i>)</a>, <a href="/wiki/Marcus_Cato" class="mw-redirect" title="Marcus Cato">Marcus Cato</a> multum de vita rustica deque praedii admininstratione temporibus Romanis scripsit. Secundum hunc librum, inter usitatissima <a href="/wiki/Animalia" title="Animalia">animalia</a> agriculturalia sunt <a href="/wiki/Bos" class="mw-redirect" title="Bos">boves</a>, <a href="/wiki/Gallus" title="Gallus">gallinas</a>, et <a href="/wiki/Ovis" class="mw-redirect" title="Ovis">oves</a>. Cato de productione <a href="/wiki/Vinum" title="Vinum">vini</a>, <a href="/wiki/Oleum" title="Oleum">olei</a>, <a href="/wiki/Instrumentum_rusticum" title="Instrumentum rusticum">instrumenti
