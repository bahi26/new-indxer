furajeră — pagină inexistentă">plantelor furajere</a> și a <a href="/w/index.php?title=Plant%C4%83_tehnic%C4%83&amp;action=edit&amp;redlink=1" class="new" title="Plantă tehnică — pagină inexistentă">plantelor tehnice</a>;</li><li><b><a href="/wiki/Apicultur%C4%83" title="Apicultură">Apicultura</a></b>, domeniu care se ocupă cu creșterea <a href="/wiki/Albin%C4%83" title="Albină">albinelor</a> pentru obținerea de <a href="/wiki/Miere" title="Miere">miere</a> și <a href="/w/index.php?title=Cear%C4%83_de_albine&amp;action=edit&amp;redlink=1" class="new" title="Ceară de albine — pagină inexistentă">ceară</a>, având și importantul rol de <a href="/wiki/Polenizare" title="Polenizare">polenizare</a> a plantelor aflate în zona de creștere a albinelor;</li><li><b><a href="/wiki/Avicultur%C4%83" title="Avicultură">Avicultura</a></b>, domeniu al agriculturii care se ocupă cu creșterea <a href="/wiki/Pas%C4%83re" title="Pasăre">păsărilor</a>;</li><li><b><a href="/wiki/Horticultur%C4%83" title="Horticultură">Horticultura</a></b>, care se ocupă cu selecționarea și creșterea <a href="/wiki/Legum%C4%83" href="/w/index.php?title=Arbust_decorativ&amp;action=edit&amp;redlink=1" class="new" title="Arbust decorativ — pagină inexistentă">decorativi</a>, <a href="/w/index.php?title=Plant%C4%83_ornamental%C4%83&amp;action=edit&amp;redlink=1" class="new" title="Plantă ornamentală — pagină inexistentă">plantelor ornamentale</a>, <a href="/w/index.php?title=Plant%C4%83_tropical%C4%83&amp;action=edit&amp;redlink=1" class="new" title="Plantă tropicală — pagină inexistentă">plantelor tropicale</a> și a <a href="/w/index.php?title=Plant%C4%83_de_ser%C4%83&amp;action=edit&amp;redlink=1" class="new" title="Plantă de seră — pagină inexistentă">plantelor de</a> <a href="/wiki/Ser%C4%83" title="Seră">seră</a>;</li><li><b><a href="/w/index.php?title=Moluscocultur%C4%83&amp;action=edit&amp;redlink=1" class="new" title="Moluscocultură — pagină inexistentă">Moluscocultura</a></b>, care se ocupă de creșterea <a href="/wiki/Molusc%C4%83" title="Moluscă">moluștelor</a>, atât <a href="/w/index.php?title=Molu%C8%99te_terestre&amp;action=edit&amp;redlink=1" class="new" title="Moluște inexistentă">Zoocultură</a></li></ul></div><p>Există și o mulțime de ramuri strict specializate ale agriculturii care au ca scop esențial producerea <a href="/wiki/Hran%C4%83" class="mw-redirect" title="Hrană">hranei</a> și [[îmbrăcăminte|i sau petrecerea timpului liber al oamenilor și a viețuitoarelor de pe lângă aceștia.</p><p>Agricultura este una dintre cele mai vechi ocupații ale omenirii, fiind intrinsec legată de <a href="/wiki/Sedentarism" title="Sedentarism">viața sedentară</a>, de când oamenii, din <a href="/w/index.php?title=Nomadism&amp;action=edit&amp;redlink=1" class="new" title="Nomadism — pagină inexistentă">nomazi</a> (axându-se pe <a href="/w/index.php?title=V%C4%83n%C4%83toare&amp;action=edit&amp;redlink=1" class="new" title="Vănătoare — pagină inexistentă">vânătoare</a>, <a href="/wiki/Pescuit" title="Pescuit">pescuit</a> și <a href="/w/index.php?title=Culeg%C4%83tor&amp;action=edit&amp;redlink=1" class="new" title="Culegător — pagină inexistentă">culegerea</a> produselor vegetale), au devenit cultivatori de plante și crescători de animale. Continua sa îmbunătățire și rafinare a fost unul din factorii esențiali ai progresului generalizat. Astăzi există oameni de știință îngust specializați, <a href="/w/index.php?title=Inventic%C4%83&amp;action=edit&amp;redlink=1" class="new" title="Inventică — pagină inexistentă">inventatori</a>, <a href="/wiki/Biologie_molecular%C4%83" title="Biologie moleculară">biologi moleculari</a>, <a href="/wiki/Genetic%C4%83" title="Genetică">geneticieni</a>, <a href="/wiki/Biochimie" title="Biochimie">biochimiști</a>, <a href="/wiki/Biofizic%C4%83" title="Biofizică">biofizicieni</a>, <a href="/wiki/Inginer" title="Inginer">ingineri</a> <a href="/w/index.php?title=Inginerie_mecanic&amp;action=edit&amp;redlink=1" class="new" title="Inginerie mecanic — pagină inexistentă">mecanici</a>, <a href="/wiki/Inginerie_chimic%C4%83" title="Inginerie chimică">chimici</a>, <a href="/wiki/Inginerie_electric%C4%83" class="mw-redirect" title="Inginerie electrică">electrici</a> și <a href="/w/index.php?title=Inginerie_electronic%C4%83&amp;action=edit&amp;redlink=1" class="new" title="Inginerie electronică — pagină inexistentă">electroniști</a> a căror unică preocupare este îmbunătățirea <a href="/wiki/Productivitate" title="Productivitate">productivității</a> și <a href="/w/index.php?title=Eficien%C8%9B%C4%83&amp;action=edit&amp;redlink=1" class="new" title="Eficiență — pagină inexistentă">eficienței</a> metodelor de cultivare a plantelor și de creștere a animalelor și a folosirii resurselor existente. Toți aceștia sunt "ocupați în agricultură".</p><p>Se consideră (conform unei <a href="/w/index.php?title=Estimare&amp;action=edit&amp;redlink=1" class="new" title="Estimare — pagină inexistentă">estimări</a> din anul <a href="/wiki/2002" title="2002">2002</a>) că <b>42%</b> din populația globului se ocupă cu agricultura, făcând-o de departe cea mai răspândită ocupație umană. Din păcate, produsele agricole contează doar ca <b>4.4%</b> (estimare din <a href="/wiki/2005" title="2005">2005</a>) din <a href="/w/index.php?title=Produs_brut_mondial&amp;action=edit&amp;redlink=1" class="new" title="Produs brut mondial — pagină inexistentă">produsul brut mondial</a>, care este calculat prin adiționarea tuturor <a href="/w/index.php?title=Produs_brut_intern&amp;action=edit&amp;redlink=1" class="new" title="Produs brut intern — pagină inexistentă">produselor brute interne</a> ale tuturor <a href="/w/index.php?title=List%C4%83_a_%C8%9B%C4%83rilor_lumii&amp;action=edit&amp;redlink=1" class="new" title="Listă a țărilor lumii — pagină inexistentă">țărilor</a>.<a rel="nofollow" class="external autonumber" href="https://www.cia.gov/library/publications/the-world-factbook/geos/xx.html">[1]</a></p><h2><span class="mw-headline" id="Istoric">Istoric</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Agricultur%C4%83&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Modifică secțiunea: Istoric">modificare</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Agricultur%C4%83&amp;action=edit&amp;section=3" title="Modifică secțiunea: Istoric">modificare sursă</a><span class="mw-editsection-bracket">]</span></span></h2><p>Agricultura este cultivarea pământului pentru a crește plante.Ea a constituit ramura principală inca de la începuturile civilizației. Această ocupație asigură cea mai mare parte a alimentelor; materiale necesare pentru îmbrăcăminte și alte materii prime pentru industrii.</p><p>Domesticirea plantelor si animalelor a avut loc acum 14000-6500 de ani si este cea mai studiata idee din preistorie.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup> Acesta a evidentiat faptului că hrana se obține nu doar prin strângerea plantelor sălbatice și vânarea animalelor ci și prin cultivarea plantelor din semințe și prin creșterea în captivitate a anumitor animale.</p><p>Jumătate din populația globului lucrează în agricultură. Există însă mari diferențe între rolul jucat de agricultură în diferite zone ale planetei. În țările aflate în curs de dezvoltare, ca de exemplu Nepalul, aproximativ 90% din populație lucrează pământul. Spre deosebire de acestea, doar aproximativ 2% din populația activă se ocupă cu agricultura, în țări industrializate precum <a href="/wiki/Regatul_Unit" class="mw-redirect" title="Regatul Unit">Regatul Unit al Marii Britanii</a> și <a href="/wiki/Statele_Unite_ale_Americii" title="Statele Unite ale Americii">Statele Unite ale Americii</a>. Cu toate acestea, datorită înaltei eficiențe și tehnicilor științifice utilizate, Statele Unite reprezintă cel mai mare exportator de produse agricole.</p><h2><span id="Studii_economice_pe_.C8.9Bar.C4.83"></span><span class="mw-headline" id="Studii_economice_pe_țară">Studii economice pe țară</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Agricultur%C4%83&amp;veaction=edit&amp;section=4" class="mw-editsection-visualeditor" title="Modifică secțiunea: Studii economice pe țară">modificare</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Agricultur%C4%83&amp;action=edit&amp;section=4" title="Modifică secțiunea: Studii economice pe țară">modificare sursă</a><span class="mw-editsection-bracket">]</span></span></h2><p>Pentru a favoriza exporturile studiile pe țară, globale sau pe sectoare sunt propuse gratuit pe site-ul lor internet, prin organisme guvernamentale. Aceste două ministere, alături de alte asociații, universități sau intreprinderi, le difuzeaza de asemenea pe site-ul <a href="/w/index.php?title=Globaltrade.net&amp;action=edit&amp;redlink=1" class="new" title="Globaltrade.net — pagină inexistentă">Globaltrade.net</a>.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup></p><p>Globaltrade le claseaza în functie de
