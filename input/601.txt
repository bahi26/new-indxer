title="Povečaj"></a></div><a href="/wiki/Kmet" title="Kmet">Kmet</a> v <a href="/wiki/Nem%C4%8Dija" title="Nemčija">Nemčiji</a> na tradicionalen način, s <a href="/wiki/Konj" class="mw-redirect" title="Konj">konjem</a> in <a href="/wiki/Plug" title="Plug">plugom</a>, obdeluje svoje <a href="/wiki/Njiva" title="Njiva">polje</a>.</div></div></div><p><b>Kmetíjstvo</b> ali <b>agrikultúra</b> je ena osnovnih in prvotnih <a href="/wiki/%C4%8Clovek" title="Človek">človekovih</a> dejavnosti. Delimo jo na <a href="/wiki/Poljedelstvo" class="mw-redirect" title="Poljedelstvo">poljedelstvo</a> in <a href="/wiki/%C5%BDivinoreja" title="Živinoreja">živinorejo</a>. <a href="/wiki/Znanost" title="Znanost">Znanstvena</a> <a href="/wiki/Veda" title="Veda">veda</a> o kmetijstvu je <a href="/wiki/Agronomija" title="Agronomija">agronomija</a>.</p><p>Poljedelstvo se je pojavilo pred približno 10.000 leti in je pomenilo velik korak v človekovem razvoju. Odkritje je omogočilo, da se je človek za stalno naselil na določenem območju. Osvajal je vedno nove površine, na katerih je gojil svoje rastline. S poljedelstvom se je zmanjšala človekova odvisnost od spontane narave. Človek je začel naravo spreminjati po svojih željah in potrebah.</p><p>Z razvojem civilizacije so države pošiljale svoje ekspedicije v svet. Z njimi so iz drugih delov sveta prihajale tudi nove živali in rastline. Evropejci so v <a href="/wiki/Amerika" class="mw-disambig" title="Amerika">Ameriko</a> pripeljali <a href="/wiki/Konj" class="mw-redirect" title="Konj">konje</a>, <a href="/wiki/Doma%C4%8De_govedo" title="Domače govedo">govejo</a> živino in <a href="/wiki/Doma%C4%8Di_pra%C5%A1i%C4%8D" title="Domači prašič">svinje</a>. Iz Amerike pa so se po vsem svetu razširila <a href="/wiki/%C5%BDito" title="Žito">žita</a>, <a href="/wiki/Je%C4%8Dmen" title="Ječmen">ječmen</a>, <a href="/wiki/Paradi%C5%BEnik" title="Paradižnik">paradižnik</a>, <a href="/wiki/Krompir" title="Krompir">krompir</a> in drugo. Tako se je obogatila prehrana vseh ljudi.</p><p>V <a href="/wiki/18._stoletje" title="18. stoletje">18. stoletju</a> so se v Evropi začela razvijati poljedelska znanja. V <a href="/wiki/19._stoletje" title="19. stoletje">19. stoletju</a> se je s tehnološko revolucijo in predvsem z mehanizacijo kmetijstva povečala tudi učinkovitost kmetijstva. Vse več dela pri obdelavi polja so opravili stroji. Za isto količino pridelka je bilo potrebno čedalje manj delovne sile. Število zaposlenih v kmetijstvu se je začelo zmanjševati.</p><p>Po koncu <a href="/wiki/Druga_svetovna_vojna" title="Druga svetovna vojna">2. svetovne vojne</a> se je z novimi načini obdelovanja zemlje, z novo tehnologijo z odkritjem <a href="/wiki/Pesticid" title="Pesticid">pesticidov</a>, <a href="/wiki/Herbicid" title="Herbicid">herbicidov</a> in z novimi načini intenzivnega kmetovanja proizvodnja hrane v razvitem svetu precej povečala. Danes je v razvitih deželah v navadi intenzivno kmetovanje z dognojevanjem in škropljenjem rastlin, z uporabo kemikalij, ki preprečujejo rast plevela in pomorijo škodljivce. Pri vzreji domačih živali se je namesto pašništva prav tako uveljavila intenzivna vzreja živali, s katero se poizkuša čim prej doseči čim večji prirast mesa.</p><p>Leta <a href="/wiki/2007" title="2007">2007</a> je <a href="/wiki/Mednarodna_organizacija_dela" title="Mednarodna organizacija dela">Mednarodna organizacija dela</a> objavila podatek, da kmetijstvo prvič od svojega nastanka ni največji sektor globalne <a href="/wiki/Ekonomija" title="Ekonomija">ekonomije</a>.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup></p><p></p><div id="toc" class="toc"><div class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Kmetijstvo&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Spremeni Sloveniji">uredi kodo</a><span class="mw-editsection-bracket">]</span></span></h2><dl><dd><div class="noprint relarticle mainarticle"><i>Glavni članek: <a href="/wiki/Kmetijstvo_in_gozdarstvo_Slovenije" title="Kmetijstvo in gozdarstvo Slovenije">Kmetijstvo in gozdarstvo Slovenije</a>.</i></div></dd></dl><p>V <a href="/wiki/Slovenija" title="Slovenija">Sloveniji</a> predstavljajo kmetijska zemljišča 32 odstotkov celotnega ozemlja, gozdovi okoli 59,9 odstotkov, 8,2 odstotka površin pa predstavljajo ostala zemljišča.</p><p>V strukturi rabe kmetijskih zemljišč zavzemajo največji delež trajni <a href="/wiki/Travnik" title="Travnik">travniki</a> in <a href="/wiki/Pa%C5%A1nik" title="Pašnik">pašniki</a> (54,6 odstotkov), sledijo jim njive (30,3 odstotkov) ter trajni nasadi (8,4 odstotkov). Slabih 7 odstotkov kmetijskih zemljišč predstavljajo ostala kmetijska zemljišča. To so opuščena zemljišča, ki se stihijsko zaraščajo.</p><p>Slovenija se uvršča med države z nadpovprečnim deležem podeželskih območij v EU. Od skupno 12 <a href="/wiki/Statisti%C4%8Dne_regije_Slovenije" title="Statistične regije Slovenije">statističnih regij</a> v Sloveniji je kar osem pretežno podeželskih, štiri pa so značilno podeželske (po kriterijih <a href="/wiki/Organizacija_za_gospodarsko_sodelovanje_in_razvoj" title="Organizacija za gospodarsko sodelovanje in razvoj">OECD</a>). Značilno podeželska območja predstavljajo 30,5 odstotka celotnega ozemlja države in zajemajo 38,5 odstotka vsega prebivalstva. Ostali del ozemlja in prebivalstva pa spada pod pretežno podeželska območja.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup></p><h2><span class="mw-headline"
