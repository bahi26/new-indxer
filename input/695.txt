Iðnaður eins og við þekkjum hann í dag varð til í <a href="/wiki/I%C3%B0nbyltingin" title="Iðnbyltingin">iðnbyltingunni</a> á <a href="/wiki/19._%C3%B6ld" class="mw-redirect" title="19. öld">19. öld</a>. Það má skipta iðnaði niður í fjóra hluta. Fyrsti hlutinn eru greinar þar sem <a href="/wiki/N%C3%A1tt%C3%BAruau%C3%B0lind" title="Náttúruauðlind">náttúruauðlindum</a> er breytt í vörur eins og <a href="/wiki/N%C3%A1muvinnsla" class="mw-redirect" title="Námuvinnsla">námuvinnsla</a>, <a href="/w/index.php?title=Sk%C3%B3garh%C3%B6gg&amp;action=edit&amp;redlink=1" class="new" title="Skógarhögg (síða er ekki enn til)">skógarhögg</a> og <a href="/wiki/Landb%C3%BAna%C3%B0ur" title="Landbúnaður">landbúnaður</a>. Annar hlutinn eru greinar þar sem hráefnum er breytt í vörur eins og <a href="/w/index.php?title=B%C3%ADlai%C3%B0na%C3%B0ur&amp;action=edit&amp;redlink=1" class="new" title="Bílaiðnaður (síða er ekki enn til)">bílaiðnaður</a> og <a href="/w/index.php?title=St%C3%A1li%C3%B0na%C3%B0ur&amp;action=edit&amp;redlink=1" class="new" title="Stáliðnaður (síða er ekki enn til)">stáliðnaður</a>. Þriðji hlutinn eru <a href="/w/index.php?title=%C3%9Ej%C3%B3nustugrein&amp;action=edit&amp;redlink=1" class="new" title="Þjónustugrein (síða er ekki enn til)">þjónustugreinar</a> eins og <a href="/wiki/Verslun" class="mw-disambig" title="Verslun">verslun</a> og <a href="/w/index.php?title=Bankastarfssemi&amp;action=edit&amp;redlink=1" class="new" title="Bankastarfssemi (síða er ekki enn til)">bankastarfssemi</a>.
