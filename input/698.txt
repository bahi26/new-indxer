class="mw-content-rtl"><div class="mw-parser-output"><p><b>תעשייה</b>, במובנה הרחב, היא קבוצה של <a href="/wiki/%D7%A2%D7%A1%D7%A7" title="עסק">עסקים</a> אשר חולקים שיטה משותפת ליצירת <a href="/wiki/%D7%A8%D7%95%D7%95%D7%97_(%D7%9B%D7%9C%D7%9B%D7%9C%D7%94)" title="רווח (כלכלה)">רווחים</a>. המונח גם משמש בהתייחסות לתחום <a href="/wiki/%D7%9B%D7%9C%D7%9B%D7%9C%D7%94" href="/wiki/%D7%94%D7%A9%D7%A7%D7%A2%D7%94" title="השקעה">השקעה</a> זמין לפני שניתן להכניס רווחים, תחום זה נקרא גם <a href="/w/index.php?title=%D7%AA%D7%A2%D7%A9%D7%99%D7%99%D7%94_%D7%9B%D7%91%D7%93%D7%94&amp;action=edit&amp;redlink=1" class="new" title="תעשייה כבדה (הדף אינו קיים)">תעשייה כבדה</a>. בעל הון העוסק בתעשייה נקרא <a href="/wiki/%D7%AA%D7%A2%D7%A9%D7%99%D7%99%D7%9F" title="תעשיין">תעשיין</a>.</p><p>התעשייה במובנה השני הפכה להיות המגזר העיקרי של ייצור ב<a href="/wiki/%D7%90%D7%99%D7%A8%D7%95%D7%A4%D7%94" title="אירופה">אירופה</a> ו<a href="/wiki/%D7%A6%D7%A4%D7%95%D7%9F_%D7%90%D7%9E%D7%A8%D7%99%D7%A7%D7%94" class="mw-redirect" title="צפון אמריקה">צפון אמריקה</a> בזמן <a href="/wiki/%D7%94%D7%9E%D7%94%D7%A4%D7%9B%D7%94_%D7%94%D7%AA%D7%A2%D7%A9%D7%99%D7%99%D7%AA%D7%99%D7%AA" title="המהפכה התעשייתית">המהפכה התעשייתית</a>. מצב זה שיבש את המסחר וה<a href="/wiki/%D7%A4%D7%99%D7%90%D7%95%D7%93%D7%9C%D7%99%D7%96%D7%9D" title="פיאודליזם">פיאודליזם</a> באמצעות התפתחויות <a href="/wiki/%D7%98%D7%9B%D7%A0%D7%95%D7%9C%D7%95%D7%92%D7%99%D7%94" title="טכנולוגיה">טכנולוגיות</a> רצופות ומהירות, כגון פיתוח ה<a href="/wiki/%D7%A7%D7%98%D7%A8_(%D7%A8%D7%9B%D7%91%D7%AA)" title="קטר (רכבת)">קטר</a>, <a href="/wiki/%D7%9E%D7%9B%D7%95%D7%A0%D7%AA_%D7%AA%D7%A4%D7%99%D7%A8%D7%94" title="מכונת תפירה">מכונת התפירה</a> המכנית והתקדמויות ייצור בתחום ה<a href="/wiki/%D7%A4%D7%9C%D7%93%D7%94" title="פלדה">פלדה</a> וה<a href="/wiki/%D7%A4%D7%97%D7%9D" title="פחם">פחם</a>. <a href="/wiki/%D7%9E%D7%93%D7%99%D7%A0%D7%94" title="מדינה">מדינות</a> תעשייתיות עברו למדיניות כלכלה <a href="/wiki/%D7%A7%D7%A4%D7%99%D7%98%D7%9C%D7%99%D7%96%D7%9D" title="קפיטליזם">קפיטליסטית</a>. <a href="/wiki/%D7%9E%D7%A1%D7%99%D7%9C%D7%AA_%D7%91%D7%A8%D7%96%D7%9C" class="mw-redirect" title="מסילת ברזל">מסילות ברזל</a> ו<a href="/wiki/%D7%A1%D7%A4%D7%99%D7%A0%D7%94" class="mw-redirect" title="ספינה">ספינות</a> <a href="/wiki/%D7%A7%D7%99%D7%98%D7%95%D7%A8" title="קיטור">קיטור</a> החלו להשתלב במהירות רבה ב<a href="/wiki/%D7%A9%D7%95%D7%A7_(%D7%9B%D7%9C%D7%9B%D7%9C%D7%94)" title="שוק (כלכלה)">שווקים</a> מרוחקים, ובכך איפשרו ל<a href="/wiki/%D7%97%D7%91%D7%A8%D7%94_%D7%A4%D7%A8%D7%98%D7%99%D7%AA" class="mw-redirect" title="חברה פרטית">חברות פרטיות</a> להתקדם לעושר ולהיקפי פעילות עצומים במושגי אותו זמן.</p><p>בכלכלה וב<a href="/wiki/%D7%AA%D7%9B%D7%A0%D7%95%D7%9F_%D7%A2%D7%A8%D7%99%D7%9D" class="mw-redirect" title="תכנון ערים">תכנון ערים</a>, "שימוש תעשייתי" הוא סוג של <a href="/wiki/%D7%A9%D7%99%D7%9E%D7%95%D7%A9_%D7%A7%D7%A8%D7%A7%D7%A2" class="mw-redirect" title="שימוש קרקע">שימוש</a> מרוכז בקרקע עליה ממוקמים עסקים כלכליים העוסקים בייצור ותפוקה.</p><div id="toc" class="toc"><div class="toctitle" lang="he" dir="rtl"><h2>תוכן עניינים</h2></div><ul><li class="toclevel-1 tocsection-1"><a href="#מרקסיזם_ותעשייה"><span class="tocnumber">1</span> <span ותעשייה&quot;">עריכת קוד מקור</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D7%AA%D7%A2%D7%A9%D7%99%D7%99%D7%94&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="עריכת פסקה: &quot;מרקסיזם ותעשייה&quot;">עריכה</a><span class="mw-editsection-bracket">]</span></span></h2><p>השליטה בתעשייה הפכה לנקודת המפתח של <a href="/wiki/%D7%9E%D7%A8%D7%A7%D7%A1%D7%99%D7%96%D7%9D" title="מרקסיזם">תאוריית המרקסיזם</a>. <a href="/wiki/%D7%AA%D7%90%D7%95%D7%A8%D7%99%D7%99%D7%AA_%D7%94%D7%A2%D7%A8%D7%9A_%D7%A9%D7%9C_%D7%94%D7%A2%D7%91%D7%95%D7%93%D7%94" title="תאוריית הערך של העבודה">תאוריית הערך של העבודה</a> אומרת כי <a href="/w/index.php?title=%D7%A2%D7%A8%D7%9A_(%D7%9B%D7%9C%D7%9B%D7%9C%D7%94)&amp;action=edit&amp;redlink=1" class="new" title="ערך (כלכלה) (הדף אינו קיים)">ערכו</a> של עצם מסוים קיים אך ורק הודות ל<b>עבודה</b> שנעשתה על מנת להביאו למצבו או צורתו הנוכחית. לעצמים בלתי מעובדים, כגון <a href="/wiki/%D7%A2%D7%A5_(%D7%97%D7%95%D7%9E%D7%A8_%D7%92%D7%9C%D7%9D)" title="עץ (חומר גלם)">עצים</a> ו<a href="/wiki/%D7%A1%D7%9C%D7%A2" title="סלע">אבנים</a>, אין בדרך כלל ערך. אך אם עץ מגולף ל<a href="/wiki/%D7%A4%D7%A1%D7%A0%D7%AA%D7%A8" title="פסנתר">פסנתר</a>, או אבנים הופכות ל<a href="/wiki/%D7%9C%D7%91%D7%A0%D7%94_(%D7%91%D7%A0%D7%99%D7%99%D7%94)" title="לבנה (בנייה)">לבנים</a> לבית, אזי לעצם הסופי יש לפתע ערך בשל העמל שנעשה.</p><p>מרקסיסטים מאמינים כי היחסים הקיימים בין המעמד ה<a href="/wiki/%D7%91%D7%95%D7%A8%D7%92%D7%A0%D7%95%D7%AA" title="בורגנות">בורגני</a> ה<a href="/wiki/%D7%A7%D7%A4%D7%99%D7%98%D7%9C%D7%99%D7%96%D7%9D" title="קפיטליזם">קפיטליסטי</a>, אלו העשירים מספיק להשקיע הון ניכר בתעשייה, לבין <a href="/wiki/%D7%9E%D7%A2%D7%9E%D7%93_%D7%94%D7%A4%D7%95%D7%A2%D7%9C%D7%99%D7%9D" title="מעמד הפועלים">מעמד הפועלים</a> אשר עובדים עבורם, הוא <a href="/wiki/%D7%A4%D7%A8%D7%96%D7%99%D7%98" class="mw-redirect" title="פרזיט">פרזיטי</a> מיסודו - כך לדוגמה, הקפיטליסט מקבל סכום <a href="/wiki/%D7%9B%D7%A1%D7%A3_(%D7%90%D7%9E%D7%A6%D7%A2%D7%99_%D7%AA%D7%A9%D7%9C%D7%95%D7%9D)" title="כסף (אמצעי תשלום)">כסף</a> הגדול בהרבה מהשקעתו המקורית, על ידי תשלום לעובדיו שהוא נמוך בהרבה מערכה האמיתי של התוצרת. לכן, מרקסיסטים תומכים בכך שעל הפועלים עצמם להחזיק ב<a href="/wiki/%D7%90%D7%9E%D7%A6%D7%A2%D7%99_%D7%99%D7%99%D7%A6%D7%95%D7%A8" title="אמצעי ייצור">אמצעי הייצור</a> ו<a href="/w/index.php?title=%D7%A0%D7%9B%D7%A1_%D7%94%D7%95%D7%9F&amp;action=edit&amp;redlink=1" class="new" title="נכס הון (הדף אינו קיים)">נכסי ההון</a>, כגון מפעלים וציוד המשמשים לייצור. במדינות <a href="/wiki/%D7%9C%D7%A0%D7%99%D7%A0%D7%99%D7%96%D7%9D" title="לניניזם">לניניסטיות</a> כגון <a
