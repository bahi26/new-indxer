class="firstHeading" lang="nds">Bueree</h1>			<div id="bodyContent" class="mw-body-content">				<div id="siteSub" class="noprint">Vun Wikipedia</div>				<div id="contentSub"></div>								<div class="mw-content-ltr"><div class="mw-parser-output"><p><b>Landweertschap</b> (de, f.) is de Anbo vun <a href="/wiki/Planten" title="Planten">Planten</a> un de <a href="/w/index.php?title=Tucht&amp;action=edit&amp;redlink=1" class="new" title="Tucht (noch nich vörhannen)">Tucht</a> vun <a href="/wiki/Veeht%C3%BC%C3%BCch" class="mw-redirect" title="Veehtüüch">Veehtüüch</a> för de <a href="/w/index.php?title=Nehrmiddel&amp;action=edit&amp;redlink=1" class="new" title="Nehrmiddel (noch nich vörhannen)">Nehrmiddelprodukschoon</a>.</p><h2><span class="mw-headline" id="Historie">Historie</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Bueree&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Ännere Afsnitt: Historie">ännern</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Bueree&amp;action=edit&amp;section=1" title="Ännere Afsnitt: Historie">Bornkood ännern</a><span class="mw-editsection-bracket">]</span></span></h2><p>Vör dat de Minschen Landweertschop dreven hebbt, sünd se al op de <a href="/w/index.php?title=Jagd&amp;action=edit&amp;redlink=1" class="new" title="Jagd (noch nich vörhannen)">Jagd</a> gahn un hebbt Planten to’n Eten sammelt. So wat vör <a href="/w/index.php?title=8000_v._Chr.&amp;action=edit&amp;redlink=1" class="new" title="8000 v. Chr. (noch nich vörhannen)">8000 v. Chr.</a> hebbt de Lü anfungen, Veehtüüch to hooln. Meist sünd se mit ehr Veehtüch in en grotet Rebeet as <a href="/w/index.php?title=Nomaden&amp;action=edit&amp;redlink=1" class="new" title="Nomaden (noch nich vörhannen)">Nomaden</a> rümtrocken.</p><p>De Anbo vun <a href="/w/index.php?title=Koorn&amp;action=edit&amp;redlink=1" class="new" title="Koorn (noch nich vörhannen)">Koorn</a> is een Markmol vun all olle <a href="/wiki/Kultur" title="Kultur">Kulturen</a>. Dat güng al in de <a href="/wiki/Steentiet" title="Steentiet">Steentiet</a> los. Bi de olen Ägypters weer dat <a href="/w/index.php?title=Garst&amp;action=edit&amp;redlink=1" class="new" title="Garst (noch nich vörhannen)">Garst</a> un <a href="/w/index.php?title=Weten&amp;action=edit&amp;redlink=1" class="new" title="Weten (noch nich vörhannen)">Weten</a>, ok bi de <a href="/wiki/Sumer" title="Sumer">Sumerers</a> un later denn bi de <a href="/w/index.php?title=Greken&amp;action=edit&amp;redlink=1" class="new" title="Greken (noch nich vörhannen)">Greken</a> un <a href="/wiki/R%C3%B6%C3%B6msch_Riek" title="Röömsch Riek">Römers</a>. In China weer dat de <a href="/w/index.php?title=Ries&amp;action=edit&amp;redlink=1" class="new" title="Ries (noch nich vörhannen)">Ries</a>, un in <a href="/wiki/Amerika" title="Amerika">Amerika</a> de <a href="/w/index.php?title=Mais&amp;action=edit&amp;redlink=1" class="new" title="Mais (noch nich vörhannen)">Mais</a>.</p><h2><span class="mw-headline" id="Oorten">Oorten</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Bueree&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Ännere Afsnitt: Oorten">ännern</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Bueree&amp;action=edit&amp;section=2" title="Ännere Afsnitt: Oorten">Bornkood ännern</a><span class="mw-editsection-bracket">]</span></span></h2><p>Na dat, wat trocken warrt, ünnerscheed wi</p><ul><li><a href="/w/index.php?title=Ackerbo&amp;action=edit&amp;redlink=1" class="new" title="Ackerbo (noch nich vörhannen)">Ackerbo</a></li><li><a href="/w/index.php?title=Veehtucht&amp;action=edit&amp;redlink=1" class="new" title="Veehtucht (noch nich vörhannen)">Veehtucht</a></li><li><a href="/w/index.php?title=Goornbo&amp;action=edit&amp;redlink=1" class="new" title="Goornbo (noch nich vörhannen)">Goornbo</a></li><li><a href="/w/index.php?title=Aaftbo&amp;action=edit&amp;redlink=1" class="new" title="Aaftbo (noch nich vörhannen)">Aaftbo</a></li></ul><p>Na de Grött vun den <a href="/wiki/Bedrief" title="Bedrief">Bedrief</a> ünnerscheed wi</p><ul><li>lütte Hööv, wo de Landweertschap blangenbi geiht</li><li><a href="/w/index.php?title=Vullerwerfshoff&amp;action=edit&amp;redlink=1" class="new" title="Vullerwerfshoff (noch nich vörhannen)">Vullerwerfshööv</a>, wo dat Inkomen tominns to’n gröttsten Deel ut de Landweertschap is</li><li><a href="/w/index.php?title=Gootshoff&amp;action=edit&amp;redlink=1" class="new" title="Gootshoff (noch nich vörhannen)">Gootshööv</a>, wo dat mehr Angestellte gifft.</li></ul><h2><span id="Bed.C3.BCden"></span><span class="mw-headline" id="Bedüden">Bedüden</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Bueree&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Ännere Afsnitt: Bedüden">ännern</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Bueree&amp;action=edit&amp;section=3" title="Ännere Afsnitt: Bedüden">Bornkood ännern</a><span Dat weern mehr as de Hälft vun all de Lü. Hüüt is dat vunwegen de <a href="/w/index.php?title=Technik&amp;action=edit&amp;redlink=1" class="new" title="Technik (noch nich vörhannen)">Technik</a> allens anners wurrn.</p><!-- NewPP limit reportParsed by mw1301Cached time: 20180307133918Cache expiry: 1900800Dynamic content: falseCPU time usage: 0.008 secondsReal time usage: 0.016 secondsPreprocessor visited node count: 34/1000000Preprocessor generated node count: 0/1500000Post‐expand include size: 0/2097152 bytesTemplate argument size: 0/2097152 bytesHighest expansion depth: 2/40Expensive parser function count: 0/500--><!--Transclusion expansion time report (%,ms,calls,template)100.00%    0.000      1 -total--></div><!-- Saved in parser cache with key ndswiki:pcache:idhash:159-0!canonical and timestamp 20180307133918 and revision id 776097 --><noscript><img src="//nds.wikipedia.org/wiki/Special:CentralAutoLogin/start?type=1x1" alt="" title="" width="1" height="1" style="border: none;
