үүннэрии</div></div></div><p><b>Агрикултуура</b>, атын аата <b>Тыа хаhаайыстыбата</b> диэн үүнээйилэри үүннэрэн уонна дьиэ кыылларын иитэн тыа малын оҥоруу. Агрикултуураны үөрэтии <a href="/w/index.php?title=%D0%90%D0%B3%D1%80%D0%B8%D0%BA%D1%83%D0%BB%D1%82%D1%83%D1%83%D1%80%D0%B0_%D2%AF%D3%A9%D1%80%D1%8D%D2%95%D1%8D&amp;action=edit&amp;redlink=1" class="new" title="Агрикултуура үөрэҕэ (маннык сирэй суох)">агрикултуура үөрэҕэ</a> диэн. Агрикультура элбэх араас көрүҥнэрдээх. Олортон бастыҥнарынан буолаллар хорутуллар сиргэ үүнээйи үүннээри уонна мэччийэр <a href="/wiki/%D0%A1%D2%AF%D3%A9%D2%BB%D2%AF" class="mw-redirect" title="Сүөһү">сүөһүнү</a> иитии. Аан дойдуга агрикултууранан 1,1&#160;млрд киhи дьарыктанар.</p><p>Агрикултуура бастыҥ оҥоhуктара: <a href="/wiki/%D0%90%D1%81" title="Ас">ас</a>, <a href="/w/index.php?title=%D0%92%D0%BE%D0%BB%D0%BE%D0%BA%D0%BD%D0%BE&amp;action=edit&amp;redlink=1" class="new" title="Волокно (маннык сирэй суох)">волокно</a>, <a href="/wiki/%D0%9E%D1%82%D1%82%D1%83%D0%BA" title="Оттук">оттук</a>, <a href="/w/index.php?title=%D0%A1%D0%B8%D0%B8%D0%BA%D1%8D%D0%B9_%D0%BC%D0%B0%D1%82%D1%8B%D1%80%D1%8B%D0%B9%D0%B0%D0%B0%D0%BB&amp;action=edit&amp;redlink=1" class="new" title="Сиикэй матырыйаал (маннык сирэй суох)">сиикэй матырыйаал</a>, көҥүллээх уонна бобуулаах <a href="/wiki/%D0%9D%D0%B0%D1%80%D0%BA%D0%BE%D1%82%D0%B8%D0%BA" title="Наркотик">наркотиктар</a>, орнаментал уонна атын дэҥ оҥоhуктар. Тиhэх сылларга үүнээйини технологик оттук быһыытынан (<a href="/w/index.php?title=%D0%91%D0%B8%D0%BE%D0%BE%D1%82%D1%82%D1%83%D0%BA&amp;action=edit&amp;redlink=1" class="new" title="Биооттук (маннык сирэй суох)">биооттук</a>), <a href="/w/index.php?title=%D0%91%D0%B8%D0%BE%D1%84%D0%B0%D1%80%D0%BC%D0%B0%D1%86%D0%B5%D0%B2%D1%82%D0%B8%D0%BA%D0%B0&amp;action=edit&amp;redlink=1" class="new" title="Биофармацевтика (маннык сирэй суох)">биофармацевтикаҕа</a> уонна <a href="/w/index.php?title=%D0%91%D0%B8%D0%BE%D0%BF%D0%BB%D0%B0%D1%81%D1%82%D0%B8%D0%BA&amp;action=edit&amp;redlink=1" class="new" title="Биопластик (маннык сирэй суох)">биопластик</a> оҥоруутугар киэҥник туттар буоллулар.</p><!-- NewPP limit reportParsed by mw1305Cached time: 20180307134501Cache expiry: 1900800Dynamic content: falseCPU time usage: 0.008 secondsReal time usage: 0.011 secondsPreprocessor visited node count: 1/1000000Preprocessor generated node count: 0/1500000Post‐expand include size: 0/2097152 bytesTemplate argument size: 0/2097152 bytesHighest expansion depth: 1/40Expensive parser function count: 0/500--><!--Transclusion expansion time report (%,ms,calls,template)100.00%    0.000      1 -total--></div><!-- Saved in parser cache with key sahwiki:pcache:idhash:3593-0!canonical and timestamp 20180307134501 and revision id 334350 --><noscript><img src="//sah.wikipedia.org/wiki/Special:CentralAutoLogin/start?type=1x1" alt="" title="" width="1" height="1" style="border: none;
