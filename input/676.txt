href="/wiki/Majandusharu" title="Majandusharu">majandusharu</a>, mille <a href="/wiki/Ettev%C3%B5te" title="Ettevõte">ettevõtted</a> (<a href="/wiki/Tehas" class="mw-redirect" title="Tehas">tehased</a>, vabrikud, <a href="/wiki/Kaevandus" title="Kaevandus">kaevandused</a>, <a href="/wiki/Elektrijaam" title="Elektrijaam">elektrijaamad</a>) valmistavad <a jaoks, toodavad <a href="/wiki/Tooraine" title="Tooraine">tooraineid</a>, <a href="/wiki/Materjal" title="Materjal">materjale</a>, <a href="/wiki/K%C3%BCtus" title="Kütus">kütust</a> ja <a href="/wiki/Energia" title="Energia">energiat</a> ning töötlevad tööstus- ja <a href="/w/index.php?title=P%C3%B5llumajandussaadus&amp;action=edit&amp;redlink=1" class="new" title="Põllumajandussaadus (pole veel kirjutatud)">põllumajandussaadusi</a>.</p><p>Tööstust jaotakse <a href="/wiki/Primaarne_sektor" title="Primaarne sektor">hankivaks</a> ja <a href="/wiki/Sekundaarne_sektor" title="Sekundaarne sektor">töötlevaks tööstuseks</a>. Aga hankiva tööstuse koosseisu kuuluvad enamasti need majandusharud (<a href="/wiki/P%C3%B5llumajandus" title="Põllumajandus">põllumajandus</a>, <a href="/wiki/Kalandus" title="Kalandus">kalandus</a>, <a href="/wiki/Metsandus" title="Metsandus">metsandus</a>, <a href="/wiki/M%C3%A4et%C3%B6%C3%B6stus" title="Mäetööstus">mäetööstus</a>), mille toodang on teiste majandusharude tooraine.</p><p>Tööstus koos <a href="/wiki/Ehitus" class="mw-redirect" title="Ehitus">ehitusega</a> kuulub <a href="/wiki/Sekundaarne_sektor" title="Sekundaarne sektor">sekundaarse sektori</a> koosseisu.</p><h2><span
