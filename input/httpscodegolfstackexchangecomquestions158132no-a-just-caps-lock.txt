(€"Aa")))Γ (                     )  Create a function g that takes a list (x:xs) and applies a function on x and xs.   · (                 )   Compose g with second argument of function in parentheses.                           Instead of x and xs, the function is called on x and the result of a recursive call of g on xs.                (€"Aa")    Check if x is 'A' or 'a'.        (?m\)              If it is, then swap the case of every char in g(xs).      §       :            Otherwise, prepend x to g(xs).</code></pre>    </div>    <div class="grid mb0 fw-wrap ai-start jc-end
